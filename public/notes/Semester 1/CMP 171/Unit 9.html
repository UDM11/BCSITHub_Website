Unit 9: Computer Security and Privacy

9.1 Computer Security

Computer security involves protecting computer systems, networks, and data from unauthorized access, attacks, or damage. It ensures the confidentiality, integrity, and availability of information.

  • Purpose: Safeguards hardware, software, and data against threats like hacking, malware, and data breaches.
  • Components: Includes access controls, encryption, and security software.
  • Examples: Firewalls, antivirus programs, and secure authentication systems.

9.2 Access Control

Access control restricts access to systems or data to authorized users only, preventing unauthorized use.

  • Methods: Uses authentication (e.g., passwords, biometrics) and authorization (e.g., role-based permissions).
  • Types: Discretionary (owner-based), mandatory (system-enforced), or role-based access control.
  • Examples: Login credentials for a computer or restricted access to sensitive files.

9.3 Protecting Against Unauthorized Access

Protecting against unauthorized access involves implementing measures to prevent intruders from accessing systems or data.

  • Techniques: Strong passwords, multi-factor authentication (MFA), and account lockout mechanisms.
  • Monitoring: Tracks login attempts and flags suspicious activities.
  • Examples: Using biometric authentication or security questions to secure user accounts.

9.4 Software Piracy

Software piracy is the unauthorized copying, distribution, or use of software, violating licensing agreements.

  • Forms: Includes illegal downloads, counterfeit software, or using single-user licenses for multiple users.
  • Impact: Causes financial losses to developers and increases security risks from unverified software.
  • Examples: Downloading cracked versions of paid software like Adobe Photoshop.

9.5 Computer Viruses (Spyware, Malware, Ransomware)

Computer viruses are malicious programs designed to harm systems, steal data, or disrupt operations.

  • Spyware: Secretly monitors user activity and collects sensitive information, e.g., keyloggers.
  • Malware: General term for malicious software, including viruses, worms, and trojans.
  • Ransomware: Locks or encrypts data, demanding payment for access, e.g., WannaCry.

9.6 Anti-Virus

Anti-virus software detects, prevents, and removes malicious programs to protect computer systems.

  • Functions: Scans files, monitors real-time activity, and updates virus definitions to combat new threats.
  • Features: Includes quarantine, removal, and system repair capabilities.
  • Examples: Norton, McAfee, and Windows Defender.

9.7 Ethical Issues

Ethical issues in computer security involve moral considerations in the use and management of technology.

  • Concerns: Includes privacy invasion, unauthorized data access, and misuse of technology for harmful purposes.
  • Principles: Respect for user privacy, transparency, and accountability in system design.
  • Examples: Ethical dilemmas in hacking (white-hat vs. black-hat) or data collection without consent.

9.8 Cyber Law

Cyber law encompasses legal frameworks governing the use of computers, networks, and digital data.

  • Purpose: Addresses cybercrimes like hacking, data theft, and online fraud.
  • Regulations: Includes laws on data protection, intellectual property, and cybercrime penalties.
  • Examples: GDPR (General Data Protection Regulation) in Europe or IT Act 2000 in India.

9.9 Network Security

Network security protects the integrity, confidentiality, and availability of data transmitted across networks.

  • Techniques: Uses firewalls, intrusion detection systems, and secure protocols like HTTPS.
  • Threats: Includes hacking, phishing, and denial-of-service (DoS) attacks.
  • Examples: Securing Wi-Fi networks with WPA3 encryption or using VPNs for secure communication.

9.10 Hardware and Software Firewall

Firewalls are security systems that monitor and control network traffic to protect systems from unauthorized access.

  • Hardware Firewall: Physical devices that filter traffic at the network level, e.g., routers with built-in firewalls.
  • Software Firewall: Programs installed on devices to control incoming and outgoing traffic, e.g., Windows Firewall.
  • Examples: Cisco ASA (hardware) or ZoneAlarm (software) firewalls.

9.11 Data and Message Security

Data and message security ensures the protection of information during storage and transmission.

  • Data Security: Protects stored data using encryption, access controls, and backups.
  • Message Security: Secures communication through protocols like SSL/TLS or end-to-end encryption.
  • Examples: Encrypting sensitive files or using WhatsAppâ€™s end-to-end encryption for messaging.

9.12 Encryption and Decryption

Encryption and decryption are processes to secure and access data by converting it into unreadable and readable formats, respectively.

  • Encryption: Converts data into a coded form using algorithms (e.g., AES, RSA) to prevent unauthorized access.
  • Decryption: Reverses encryption to make data readable using a key.
  • Examples: HTTPS websites use encryption to secure data transmission, and encrypted USB drives require keys for decryption.
