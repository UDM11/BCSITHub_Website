Unit 2: Functions, Limit and Continuity

2.1 Constants and Variables, Concept of Functions

Constants, variables, and functions form the foundation of mathematical modeling, essential for describing relationships in mathematics, computer science, and economics.

  • Constants: Fixed values that do not change, such as π or 5, used as reference points in equations or models.
  • Variables: Symbols representing quantities that can change, such as x or y, allowing flexibility in mathematical expressions.
  • Functions: A rule that assigns each input (from a set called the domain) to exactly one output (in a set called the range). Denoted as f(x), e.g., f(x) = x² maps x to x².
  • Notation and Representation: Functions are expressed as equations, tables, or graphs, with f(x) indicating the output for input x (e.g., f(2) = 4 for f(x) = x²).

Understanding these concepts enables modeling of real-world relationships, such as cost functions in business.

2.2 Domain and Range of a Function

The domain and range define the scope and output of a function, critical for understanding its behavior and applicability.

  • Domain: The set of all possible input values (x) for which the function is defined. For example, for f(x) = 1/x, the domain is all real numbers except x = 0.
  • Range: The set of all possible output values (f(x)) produced by the function. For f(x) = x², the range is all non-negative real numbers (y ≥ 0).
  • Determining Domain: Identify restrictions, such as division by zero or square roots of negative numbers (e.g., f(x) = √x has domain x ≥ 0).
  • Determining Range: Analyze the function’s behavior or graph to find all possible outputs, often using algebraic methods or testing values.

Domain and range clarify the function’s applicability in fields like programming and economics.

2.3 Types of Functions (Algebraic, Logarithmic, Trigonometric, Exponential)

Different types of functions serve distinct purposes in modeling phenomena, each with unique properties and applications.

  • Algebraic Functions: Include polynomials (e.g., f(x) = x² + 2x), rational functions (e.g., f(x) = 1/x), and root functions (e.g., f(x) = √x). Used in optimization and data analysis.
  • Logarithmic Functions: Of the form f(x) = log_b(x), where b is the base (e.g., f(x) = ln(x)). Model growth rates or scales, common in information theory and economics.
  • Trigonometric Functions: Include sin(x), cos(x), and tan(x), used to model periodic phenomena like waves or oscillations in signal processing and engineering.
  • Exponential Functions: Of the form f(x) = a^x, where a > 0 (e.g., f(x) = e^x). Model rapid growth or decay, such as compound interest or radioactive decay.

These function types enable diverse modeling in scientific and economic applications.

2.4 Graphic Representation

Graphic representation visualizes functions, revealing their behavior, trends, and key features through plots and graphs.

  • Cartesian Plane: Functions are plotted on a coordinate system with x (input) and y (output) axes, where each point (x, f(x)) represents a value of the function.
  • Key Features: Identify intercepts (where f(x) = 0 or x = 0), vertices (for quadratics), asymptotes (for rational functions), and periodicity (for trigonometric functions).
  • Graphing Techniques: Use tables of values, symmetry, or transformations (e.g., shifts, stretches) to sketch graphs accurately, or employ tools like graphing calculators.
  • Applications: Graphs visualize trends in data analysis, such as profit functions in business or signal patterns in computing.

Graphic representation aids in interpreting and communicating functional relationships effectively.

2.5 Application of Functions to Business and Economics

Functions model relationships in business and economics, enabling analysis and prediction of financial and operational outcomes.

  • Cost Functions: Represent total costs as a function of output, e.g., C(q) = 100 + 5q, where q is quantity produced, used to optimize production.
  • Revenue and Profit Functions: Revenue R(q) = p·q (price times quantity) and profit P(q) = R(q) - C(q) help determine break-even points or maximum profit.
  • Demand Functions: Model quantity demanded as a function of price, e.g., q = 100 - 2p, used to predict consumer behavior and set pricing strategies.
  • Exponential Models: Used for compound interest (A = P(1 + r/n)^(nt)) or depreciation, critical for financial planning and investment analysis.

Functions provide quantitative tools for decision-making in business and economic contexts.

2.6 Limit of a Function, Properties of Limit

The limit of a function describes its behavior as the input approaches a specific value, foundational for calculus and analytical modeling.

  • Limit Definition: The limit of f(x) as x approaches a is L, written lim(x→a) f(x) = L, if f(x) gets arbitrarily close to L as x nears a.
  • Properties of Limits: For functions f and g, and constants c, k: lim(x→a) [cf(x)] = c·lim(x→a) f(x); lim(x→a) [f(x) + g(x)] = lim(x→a) f(x) + lim(x→a) g(x); lim(x→a) [f(x)·g(x)] = lim(x→a) f(x)·lim(x→a) g(x).
  • Evaluation Techniques: Use substitution, factoring, or rationalization to compute limits, especially when direct substitution yields indeterminate forms like 0/0.
  • Applications: Limits model instantaneous rates (e.g., velocity) or asymptotic behavior in algorithms and economic forecasting.

Limits provide a precise way to analyze function behavior near specific points.

2.7 Limits of Polynomial and Rational Functions

Limits of polynomial and rational functions are computed using algebraic techniques, critical for understanding their behavior in mathematical models.

  • Polynomial Functions: For f(x) = a_n x^n + … + a_0, lim(x→a) f(x) = f(a) by direct substitution, as polynomials are defined everywhere (e.g., lim(x→2) (x² + 3) = 7).
  • Rational Functions: For f(x) = p(x)/q(x), if q(a) ≠ 0, lim(x→a) f(x) = p(a)/q(a). If q(a) = 0, factor or simplify to resolve indeterminate forms (e.g., lim(x→1) [(x²-1)/(x-1)] = lim(x→1) (x+1) = 2).
  • Indeterminate Forms: Forms like 0/0 require algebraic manipulation, such as canceling common factors or using L’Hôpital’s rule for advanced cases.
  • Applications: Used in optimization problems or modeling cost/revenue functions where precise behavior at specific points is needed.

These limits ensure accurate analysis of polynomial and rational models in various fields.

2.8 Limits at Infinity

Limits at infinity describe a function’s behavior as the input grows arbitrarily large or small, useful for analyzing long-term trends.

  • Definition: lim(x→∞) f(x) = L means f(x) approaches L as x increases without bound; similarly for lim(x→-∞) f(x).
  • Polynomial Functions: For f(x) = a_n x^n + …, lim(x→∞) f(x) depends on the leading term: if n is even and a_n > 0, lim(x→∞) = ∞; if odd and a_n < 0, lim(x→∞) = -∞.
  • Rational Functions: For f(x) = p(x)/q(x), compare degrees of numerator and denominator: if deg(p) < deg(q), lim(x→∞) = 0; if equal, lim(x→∞) = leading coefficient ratio; if deg(p) > deg(q), limit is ±∞.
  • Applications: Model long-term behavior in economics (e.g., market saturation) or algorithm efficiency (e.g., time complexity).

Limits at infinity reveal asymptotic trends critical for strategic planning and analysis.

2.9 Continuity, Continuity at a Point

Continuity describes a function’s smoothness, ensuring no breaks or jumps, vital for reliable mathematical and computational models.

  • Definition: A function f is continuous at x = a if: (1) f(a) is defined, (2) lim(x→a) f(x) exists, and (3) lim(x→a) f(x) = f(a).
  • Types of Discontinuities: Removable (e.g., f(x) = (x²-1)/(x-1) at x = 1), jump (e.g., piecewise functions with different limits), or infinite (e.g., f(x) = 1/x at x = 0).
  • Continuity on an Interval: A function is continuous on [a, b] if it is continuous at every point in the interval, ensuring predictable behavior.
  • Applications: Ensures stable algorithms, reliable simulations, and valid economic models where abrupt changes are unrealistic.

Continuity guarantees consistent and predictable function behavior in practical applications.

2.10 Business Application of Limit

Limits have practical applications in business, enabling precise analysis of trends, costs, and performance metrics.

  • Marginal Analysis: Limits calculate marginal cost or revenue (e.g., lim(Δq→0) [C(q+Δq) - C(q)]/Δq), used to optimize production levels.
  • Break-even Points: Limits help analyze behavior near critical points, such as where cost equals revenue, refining financial strategies.
  • Market Saturation: Limits at infinity model long-term demand (e.g., lim(q→∞) D(q) = 0 for diminishing demand), aiding strategic planning.
  • Forecasting Trends: Limits analyze asymptotic behavior of sales or profit functions, predicting steady-state outcomes for investment decisions.

Limits provide quantitative precision for optimizing business operations and planning.
