Unit 2: Elements of C

2.1 C Character Set

The C character set consists of the characters that can be used to form valid C programs. These characters are used to create tokens, strings, and other elements in C.

  • Letters: A-Z, a-z (both uppercase and lowercase).
  • Digits: 0-9.
  • Special Characters: Symbols like +, -, *, /, %, &, |, etc.
  • White Spaces: Spaces, tabs, newlines, etc.

2.2 Tokens

Tokens are the smallest individual units in a C program, recognized by the compiler.

  • Keywords: Reserved words with specific meanings (e.g., int, for, if).
  • Identifiers: User-defined names for variables, functions, etc. (e.g., main, sum).
  • Constants: Fixed values like 10, 3.14, or 'A'.
  • Operators: Symbols for operations (e.g., +, -, *, /).
  • Punctuators: Symbols like ;, ,, {}, (), etc., for syntax.
  • Example: 
    int main() {
    int x = 5;
}
    Tokens: int, main, (, ), {, int, x, =, 5, ;, }.

2.3 Escape Sequence

Escape sequences are special character combinations starting with a backslash (\) used to represent characters that are difficult to include directly in code.

  • Common Escape Sequences:
    • \n: Newline
    • \t: Tab
    • \": Double quote
    • \\ : Backslash
    • \0: Null character
  • Example: 
    #include <stdio.h>
int main() {
    printf("Hello\nWorld\t!\\");
    return 0;
}
    Output: 
    Hello
World   !

2.4 Variables Declaration and Initialization

Variables are named memory locations used to store data. Declaration specifies the variable's type, and initialization assigns an initial value.

  • Declaration: Specifies the data type and name. 
    int age;
float salary;
  • Initialization: Assigns a value at the time of declaration. 
    int age = 25;
float salary = 50000.50;
  • Rules: Variable names must start with a letter or underscore, contain no spaces, and avoid reserved keywords.

2.5 Data Types

Data types define the type and size of data a variable can hold.

2.5.1 Basic, Derived, and User Defined

  • Basic Data Types:
    • int: Integer (e.g., 10, -5).
    • float: Single-precision floating-point (e.g., 3.14).
    • double: Double-precision floating-point (e.g., 3.14159).
    • char: Single character (e.g., 'A').
  • Derived Data Types:
    • Array: Collection of elements of the same type (e.g., int arr[5]).
    • Pointer: Stores memory address (e.g., int *ptr).
    • Function: A block of code with a return type.
  • User-Defined Data Types:
    • struct: Groups different data types. 
      struct student {
    int id;
    char name[50];
};
    • union: Shares memory for different data types.
    • enum: Defines a set of named integer constants.

2.6 Constants/Literals

Constants are fixed values that cannot be modified during program execution.

  • Types:
    • Integer Literals: Whole numbers (e.g., 10, -5).
    • Floating-Point Literals: Decimal numbers (e.g., 3.14, -0.001).
    • Character Literals: Single characters in single quotes (e.g., 'A').
    • String Literals: Sequence of characters in double quotes (e.g., "Hello").
  • Defining Constants: 
    #define MAX 100
const int MIN = 10;

2.7 Expressions

An expression is a combination of variables, constants, and operators that produces a value.

  • Types:
    • Arithmetic: e.g., a + b, x * y.
    • Relational: e.g., a > b, x == y.
    • Logical: e.g., (a > b) && (x < y).
    • Assignment: e.g., x = 5.
  • Example: 
    int x = 5, y = 3;
int z = x + y * 2; // z = 11 (follows operator precedence)

2.8 Statements

Statements are complete instructions in a C program that perform actions.

  • Types:
    • Expression Statements: e.g., x = 5;.
    • Compound Statements: Group of statements in {}.
    • Control Statements: e.g., if, for, while.
    • Jump Statements: e.g., break, continue, return.
  • Example: 
    if (x > 0) {
    printf("Positive");
}

2.9 Comments

Comments are non-executable text used to explain code, ignored by the compiler.

  • Single-Line Comments: Start with //. 
    // This is a single-line comment
int x = 5;
  • Multi-Line Comments: Enclosed in /* */. 
    /* This is a
   multi-line comment */
int y = 10;
  • Purpose: Improve code readability and maintenance.
