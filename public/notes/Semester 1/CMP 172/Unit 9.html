Unit 9: Structure and Union

9.1 Introduction

Structures and unions are user-defined data types in C that allow grouping of variables of different types under a single name, enabling organized data management.

  • Structure: Combines multiple variables, each retaining its own memory.
  • Union: Combines variables that share the same memory location.
  • Purpose: Simplifies handling complex data, like records or configurations.

9.2 Declaration and Initialization of Structure

A structure is declared using the struct keyword, and variables can be initialized during or after declaration.

  • Declaration Syntax: 
    struct structure_name {
    data_type member1;
    data_type member2;
    // ...
};
  • Initialization: At declaration or later using dot operator. 
    #include <stdio.h>
struct student {
    int id;
    char name[20];
    float marks;
};
int main() {
    // Initialization at declaration
    struct student s1 = {101, "Alice", 85.5};
    // Initialization after declaration
    struct student s2;
    s2.id = 102;
    strcpy(s2.name, "Bob");
    s2.marks = 90.0;
    printf("ID: %d, Name: %s, Marks: %.1f\n", s1.id, s1.name, s1.marks);
    return 0;
}

9.3 Structure Variable

Structure variables are instances of a structure, used to store data for its members.

  • Declaration: struct structure_name variable_name;
  • Access: Use dot operator (.) to access members. 
    #include <stdio.h>
struct point {
    int x;
    int y;
};
int main() {
    struct point p1 = {3, 4};
    printf("Point: (%d, %d)\n", p1.x, p1.y);
    return 0;
}

9.4 Array of Structure

An array of structures stores multiple structure instances, useful for managing collections of records.

  • Syntax: struct structure_name array_name[size];
  • Example: 
    #include <stdio.h>
struct student {
    int id;
    char name[20];
};
int main() {
    struct student students[3] = {
        {1, "Alice"},
        {2, "Bob"},
        {3, "Charlie"}
    };
    for (int i = 0; i < 3; i++) {
        printf("ID: %d, Name: %s\n", students[i].id, students[i].name);
    }
    return 0;
}

9.5 Passing Structure to Function

Structures can be passed to functions by value or by reference (using pointers).

  • Pass by Value: Copies the structure, changes don’t affect original. 
    #include <stdio.h>
struct point {
    int x;
    int y;
};
void printPoint(struct point p) {
    printf("Point: (%d, %d)\n", p.x, p.y);
}
int main() {
    struct point p1 = {5, 10};
    printPoint(p1);
    return 0;
}
  • Pass by Reference: Uses pointer to modify original structure. 
    #include <stdio.h>
struct point {
    int x;
    int y;
};
void modifyPoint(struct point *p) {
    p->x = 20;
    p->y = 30;
}
int main() {
    struct point p1 = {5, 10};
    modifyPoint(&p1);
    printf("Modified Point: (%d, %d)\n", p1.x, p1.y);
    return 0;
}

9.6 Passing Array of Structure to Function

An array of structures is passed to functions by reference, allowing modifications to the original data.

  • Example: 
    #include <stdio.h>
struct student {
    int id;
    char name[20];
};
void printStudents(struct student s[], int size) {
    for (int i = 0; i < size; i++) {
        printf("ID: %d, Name: %s\n", s[i].id, s[i].name);
    }
}
int main() {
    struct student students[2] = {{1, "Alice"}, {2, "Bob"}};
    printStudents(students, 2);
    return 0;
}

9.7 Nested Structure

A nested structure contains another structure as a member, enabling hierarchical data organization.

  • Syntax: Define one structure inside another or separately. 
    #include <stdio.h>
struct date {
    int day;
    int month;
    int year;
};
struct employee {
    int id;
    char name[20];
    struct date dob;
};
int main() {
    struct employee emp = {101, "Alice", {15, 5, 1990}};
    printf("ID: %d, Name: %s, DOB: %d/%d/%d\n", 
           emp.id, emp.name, emp.dob.day, emp.dob.month, emp.dob.year);
    return 0;
}

9.8 Pointer to Structure

A pointer to a structure points to the memory address of a structure variable, accessed using the arrow operator (->).

  • Syntax: struct structure_name *pointer_name;
  • Example: 
    #include <stdio.h>
struct point {
    int x;
    int y;
};
int main() {
    struct point p1 = {5, 10};
    struct point *ptr = &p1;
    printf("Point: (%d, %d)\n", ptr->x, ptr->y);
    return 0;
}
  • Note: -> is equivalent to (*ptr).member.

9.10 Union vs Structure

Unions and structures both group variables, but differ in memory allocation and usage.

  • Union Declaration: 
    union data {
    int i;
    float f;
    char c;
};
  • Comparison:
    • Structure:
      • Each member has its own memory.
      • Size is sum of all members’ sizes (plus padding).
      • All members can hold values simultaneously.
    • Union:
      • All members share the same memory.
      • Size is that of the largest member.
      • Only one member can hold a value at a time.
  • Example: 
    #include <stdio.h>
union data {
    int i;
    float f;
};
int main() {
    union data d;
    d.i = 10;
    printf("Integer: %d\n", d.i);
    d.f = 3.14;
    printf("Float: %.2f\n", d.f); // i is overwritten
    return 0;
}
  • Use Case: Unions save memory when only one member is needed at a time; structures are used for multiple simultaneous values.
