Unit 3: Input and Output

3.1 Unformatted and Formatted I/O

Input and Output (I/O) operations in C allow programs to interact with users by reading input and displaying output. I/O functions are categorized as unformatted and formatted.

  • Unformatted I/O: Handles simple input/output without specific formatting, typically for characters or strings.
    • Examples: getchar(), putchar(), gets(), puts().
    • Use Case: Reading/writing single characters or strings without format control.
    • Example: 
      #include <stdio.h>
int main() {
    char ch = getchar();
    putchar(ch);
    return 0;
}
  • Formatted I/O: Allows precise control over input/output format using format specifiers.
    • Examples: scanf(), printf().
    • Use Case: Reading/writing data with specific formats (e.g., integers, floats).
    • Example: 
      #include <stdio.h>
int main() {
    int num;
    printf("Enter a number: ");
    scanf("%d", &num);
    printf("You entered: %d\n", num);
    return 0;
}

3.2 Conversion Specifier

Conversion specifiers are used in formatted I/O functions like printf() and scanf() to define the type and format of data being read or written.

  • Common Conversion Specifiers:
    • %d: Integer (decimal).
    • %f: Float.
    • %c: Character.
    • %s: String.
    • %x: Hexadecimal.
    • %o: Octal.
  • Format Modifiers:
    • %ld: Long integer.
    • %lf: Double.
    • %.2f: Float with 2 decimal places.
  • Example: 
    #include <stdio.h>
int main() {
    float pi = 3.14159;
    printf("Pi = %.2f\n", pi); // Outputs: Pi = 3.14
    return 0;
}

3.3 Character Input and Output

Character I/O functions handle individual characters, useful for processing text or single-key inputs.

  • Input Functions:
    • getchar(): Reads a single character from standard input.
    • getc(): Reads a character from a specified stream (e.g., stdin).
    • fgetc(): Similar to getc(), reads from a file stream.
  • Output Functions:
    • putchar(): Writes a single character to standard output.
    • putc(): Writes a character to a specified stream.
    • fputc(): Writes a character to a file stream.
  • Example: 
    #include <stdio.h>
int main() {
    char ch;
    printf("Enter a character: ");
    ch = getchar();
    printf("You entered: ");
    putchar(ch);
    printf("\n");
    return 0;
}

3.4 I/O Operations

I/O operations in C involve reading from and writing to various streams, such as standard input/output (stdin/stdout) or files.

  • Standard I/O:
    • stdin: Standard input, typically the keyboard.
    • stdout: Standard output, typically the console.
    • stderr: Standard error, for error messages.
  • File I/O: Reading from and writing to files using functions like fopen(), fread(), fwrite(), fclose().
  • Buffering:
    • Line Buffering: Output is stored until a newline is encountered (e.g., printf with \n).
    • Full Buffering: Output is stored until the buffer is full or flushed.
    • No Buffering: Immediate output (e.g., stderr).
  • Example (File I/O): 
    #include <stdio.h>
int main() {
    FILE *fp;
    fp = fopen("example.txt", "w");
    if (fp != NULL) {
        fputs("Hello, File!\n", fp);
        fclose(fp);
    }
    return 0;
}
