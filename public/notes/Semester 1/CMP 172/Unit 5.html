Unit 5: Control Statement

5.1 Introduction to Control Structure

Control structures in C determine the flow of program execution. They allow a program to make decisions, repeat actions, or execute statements in a specific order.

  • Types:
    • Sequential: Executes statements in order.
    • Decision-Making: Chooses between alternatives.
    • Iteration: Repeats statements.
  • Purpose: Enables dynamic and flexible program behavior.

5.2 Sequential Control Statement

Sequential control statements execute in the order they appear in the program, from top to bottom.

  • Characteristics: No conditions or loops; each statement runs once.
  • Example: 
    #include <stdio.h>
int main() {
    int a = 5, b = 10;
    int sum = a + b;
    printf("Sum: %d\n", sum);
    return 0;
}
    Output: Sum: 15

5.3 Decision Making Statement

Decision-making statements allow a program to execute different code blocks based on conditions.

5.3.1 if, if else, if else if, nested if, and switch case

  • if Statement: Executes a block if a condition is true. 
    #include <stdio.h>
int main() {
    int x = 10;
    if (x > 5) {
        printf("x is greater than 5\n");
    }
    return 0;
}
  • if else Statement: Executes one block if true, another if false. 
    #include <stdio.h>
int main() {
    int x = 3;
    if (x > 5) {
        printf("x is greater than 5\n");
    } else {
        printf("x is not greater than 5\n");
    }
    return 0;
}
  • if else if Statement: Checks multiple conditions sequentially. 
    #include <stdio.h>
int main() {
    int marks = 75;
    if (marks >= 90) {
        printf("Grade A\n");
    } else if (marks >= 70) {
        printf("Grade B\n");
    } else {
        printf("Grade C\n");
    }
    return 0;
}
  • Nested if Statement: if statements within if statements. 
    #include <stdio.h>
int main() {
    int x = 10, y = 5;
    if (x > 0) {
        if (y > 0) {
            printf("Both are positive\n");
        }
    }
    return 0;
}
  • switch case Statement: Selects one of many code blocks based on a variableâ€™s value. 
    #include <stdio.h>
int main() {
    int day = 3;
    switch (day) {
        case 1: printf("Monday\n"); break;
        case 2: printf("Tuesday\n"); break;
        case 3: printf("Wednesday\n"); break;
        default: printf("Invalid day\n");
    }
    return 0;
}
    Note: break prevents fall-through to subsequent cases.

5.4 Iteration (Loop) Statement

Iteration statements (loops) repeat a block of code until a condition is met.

5.4.1 for, while, do while, nested loops

  • for Loop: Repeats for a fixed number of iterations. 
    #include <stdio.h>
int main() {
    for (int i = 1; i <= 5; i++) {
        printf("%d ", i);
    }
    return 0;
}
    Output: 1 2 3 4 5
  • while Loop: Repeats while a condition is true. 
    #include <stdio.h>
int main() {
    int i = 1;
    while (i <= 5) {
        printf("%d ", i);
        i++;
    }
    return 0;
}
    Output: 1 2 3 4 5
  • do while Loop: Executes at least once, then repeats while true. 
    #include <stdio.h>
int main() {
    int i = 1;
    do {
        printf("%d ", i);
        i++;
    } while (i <= 5);
    return 0;
}
    Output: 1 2 3 4 5
  • Nested Loops: Loops within loops for complex iterations. 
    #include <stdio.h>
int main() {
    for (int i = 1; i <= 3; i++) {
        for (int j = 1; j <= 2; j++) {
            printf("(%d,%d) ", i, j);
        }
    }
    return 0;
}
    Output: (1,1) (1,2) (2,1) (2,2) (3,1) (3,2)

5.5 Break and Continue

Break and continue statements alter the flow of loops or switch cases.

  • break: Exits the innermost loop or switch case. 
    #include <stdio.h>
int main() {
    for (int i = 1; i <= 5; i++) {
        if (i == 3) {
            break;
        }
        printf("%d ", i);
    }
    return 0;
}
    Output: 1 2
  • continue: Skips the rest of the current iteration and proceeds to the next. 
    #include <stdio.h>
int main() {
    for (int i = 1; i <= 5; i++) {
        if (i == 3) {
            continue;
        }
        printf("%d ", i);
    }
    return 0;
}
    Output: 1 2 4 5
