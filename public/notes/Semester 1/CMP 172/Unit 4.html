Unit 4: Operators and Expression

4.1 Operator, Operand, and Expression

Operators are symbols that perform operations on operands, which are variables or values. An expression is a combination of operators and operands that evaluates to a single value.

  • Operator: Symbols like +, -, *, /, etc.
  • Operand: Values or variables operated on, e.g., 5, x.
  • Expression: Combines operators and operands, e.g., x + 5.
  • Example: 
    int x = 10, y = 5;
int z = x + y; // x + y is an expression

4.2 Types of Operators

C provides various types of operators to perform different operations.

4.2.1 Arithmetic Operator

Arithmetic operators perform mathematical calculations.

  • Operators:
    • +: Addition
    • -: Subtraction
    • *: Multiplication
    • /: Division
    • %: Modulus (remainder)
  • Example: 
    #include <stdio.h>
int main() {
    int a = 10, b = 3;
    printf("Sum: %d\n", a + b); // 13
    printf("Mod: %d\n", a % b); // 1
    return 0;
}

4.2.2 Relational Operator

Relational operators compare two values and return true (1) or false (0).

  • Operators:
    • ==: Equal to
    • !=: Not equal to
    • >: Greater than
    • <: Less than
    • >=: Greater than or equal to
    • <=: Less than or equal to
  • Example: 
    #include <stdio.h>
int main() {
    int a = 5, b = 3;
    printf("a > b: %d\n", a > b); // 1 (true)
    printf("a == b: %d\n", a == b); // 0 (false)
    return 0;
}

4.2.3 Logical or Boolean Operator

Logical operators combine conditions to produce a single boolean result.

  • Operators:
    • &&: Logical AND (true if both operands are true)
    • ||: Logical OR (true if at least one operand is true)
    • !: Logical NOT (inverts the truth value)
  • Example: 
    #include <stdio.h>
int main() {
    int a = 1, b = 0;
    printf("a && b: %d\n", a && b); // 0 (false)
    printf("!a: %d\n", !a); // 0 (false)
    return 0;
}

4.2.4 Assignment Operator

Assignment operators assign values to variables, often combining with other operations.

  • Operators:
    • =: Simple assignment
    • +=: Add and assign
    • -=: Subtract and assign
    • *=: Multiply and assign
    • /=: Divide and assign
    • %=: Modulus and assign
  • Example: 
    #include <stdio.h>
int main() {
    int x = 10;
    x += 5; // x = x + 5
    printf("x: %d\n", x); // 15
    return 0;
}

4.2.5 Bitwise Operator

Bitwise operators perform operations on binary representations of integers.

  • Operators:
    • &: Bitwise AND
    • |: Bitwise OR
    • ^: Bitwise XOR
    • ~: Bitwise NOT
    • <<: Left shift
    • >>: Right shift
  • Example: 
    #include <stdio.h>
int main() {
    int a = 5, b = 3; // a: 0101, b: 0011
    printf("a & b: %d\n", a & b); // 0001 (1)
    printf("a << 1: %d\n", a << 1); // 1010 (10)
    return 0;
}

4.2.6 Prefix and Postfix Increment and Decrement Operator

Increment (++) and decrement (--) operators modify a variable's value by 1, with prefix and postfix forms affecting evaluation order.

  • Prefix: Changes value before use (e.g., ++x).
  • Postfix: Changes value after use (e.g., x++).
  • Example: 
    #include <stdio.h>
int main() {
    int x = 5, y;
    y = ++x; // x: 6, y: 6
    printf("x: %d, y: %d\n", x, y);
    y = x++; // y: 6, x: 7
    printf("x: %d, y: %d\n", x, y);
    return 0;
}

4.2.7 Ternary Operator

The ternary operator (?:) is a concise conditional operator that evaluates an expression based on a condition.

  • Syntax: condition ? expression1 : expression2
  • Example: 
    #include <stdio.h>
int main() {
    int a = 10, b = 5;
    int max = (a > b) ? a : b;
    printf("Max: %d\n", max); // 10
    return 0;
}

4.2.8 Special Operators (sizeof and comma)

Special operators perform unique tasks in C.

  • sizeof: Returns the size (in bytes) of a variable or data type. 
    #include <stdio.h>
int main() {
    printf("Size of int: %zu bytes\n", sizeof(int));
    return 0;
}
  • Comma (,): Separates expressions, evaluated left to right, with the value of the last expression returned. 
    #include <stdio.h>
int main() {
    int x = (2, 3, 4);
    printf("x: %d\n", x); // 4
    return 0;
}

4.3 Evaluation of Expression

Expressions are evaluated based on operator precedence and associativity, producing a single value.

  • Steps:
    • Identify operators and operands.
    • Apply precedence rules (e.g., * before +).
    • Resolve ties using associativity (e.g., left-to-right for +).
  • Example: 
    #include <stdio.h>
int main() {
    int x = 5 + 3 * 2; // * has higher precedence
    printf("x: %d\n", x); // 11 (3 * 2 = 6, then 5 + 6)
    return 0;
}

4.4 Operator Precedence and Associativity

Operator precedence determines the order of operations, while associativity resolves the order for operators of equal precedence.

  • Precedence: Higher precedence operators are evaluated first (e.g., * and / before + and -).
  • Associativity: Left-to-right (e.g., for +, -) or right-to-left (e.g., for =).
  • Common Precedence (high to low):
    • (): Parentheses
    • ++, -- (prefix), sizeof
    • *, /, %
    • +, -
    • <, >, <=, >=
    • ==, !=
    • &&
    • ||
    • ?:
    • =, +=, -=, etc.
  • Example: 
    #include <stdio.h>
int main() {
    int x = 5 + 3 * 2 > 10 && 1; // * first, then +, then >, then &&
    printf("x: %d\n", x); // 0 (false)
    return 0;
}
