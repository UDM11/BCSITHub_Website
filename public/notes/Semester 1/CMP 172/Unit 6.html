Unit 6: Arrays and String

6.1 Introduction to Array

An array is a collection of elements of the same data type stored in contiguous memory locations, accessed using a common name and index.

  • Purpose: Simplifies handling multiple values of the same type.
  • Characteristics: Fixed size, indexed from 0.
  • Example: Storing 5 integers in an array instead of 5 separate variables.

6.2 Declaration

Array declaration specifies the data type, array name, and size.

  • Syntax: data_type array_name[size];
  • Examples: 
    int numbers[5]; // Array of 5 integers
float scores[10]; // Array of 10 floats
  • Note: Size must be a positive integer constant at declaration.

6.3 Memory Representation of Array

Arrays are stored in contiguous memory locations, with each element occupying a fixed size based on its data type.

  • Memory Layout: For int arr[3], if arr[0] is at address 1000 and int size is 4 bytes, arr[1] is at 1004, arr[2] at 1008.
  • Access: Index calculates address as: base_address + (index * size_of_data_type).
  • Example: 
    #include <stdio.h>
int main() {
    int arr[3] = {10, 20, 30};
    printf("Address of arr[0]: %p\n", &arr[0]);
    printf("Address of arr[1]: %p\n", &arr[1]);
    return 0;
}
    Output shows consecutive addresses.

6.4 Initialization of Array

Arrays can be initialized at declaration or later using loops.

  • At Declaration: 
    int arr[5] = {1, 2, 3, 4, 5}; // Full initialization
int arr2[5] = {1, 2}; // Partial: arr2[2..4] = 0
int arr3[] = {1, 2, 3}; // Size inferred as 3
  • Using Loops: 
    #include <stdio.h>
int main() {
    int arr[5];
    for (int i = 0; i < 5; i++) {
        arr[i] = i + 1;
    }
    return 0;
}
  • Note: Uninitialized arrays may contain garbage values.

6.5 Types of Arrays

6.5.1 Single Dimensional and Multidimensional

  • Single Dimensional Array: A linear array with one index. 
    int arr[5] = {10, 20, 30, 40, 50};
  • Multidimensional Array: Arrays with multiple indices, e.g., 2D (matrix) or 3D.
    • 2D Array Syntax: data_type array_name[rows][columns];
    • Example: 
      #include <stdio.h>
int main() {
    int matrix[2][3] = {{1, 2, 3}, {4, 5, 6}};
    printf("Element at [1][2]: %d\n", matrix[1][2]); // 6
    return 0;
}
    • Memory: Stored row-major order (consecutive elements of a row are contiguous).

6.7 Character Array and Strings

A character array stores characters, and a string is a character array terminated by a null character (\0).

  • Character Array: 
    char arr[5] = {'H', 'e', 'l', 'l', 'o'};
  • String: 
    char str[6] = "Hello"; // Includes \0 automatically
  • Difference: Strings are null-terminated; character arrays may not be.

6.8 Reading and Writing Strings

Strings are read and written using specific I/O functions.

  • Reading Strings:
    • scanf("%s", str): Reads until whitespace, no & needed for arrays.
    • gets(str): Reads entire line (unsafe, deprecated).
    • fgets(str, size, stdin): Safer, reads up to size-1 characters or newline.
    • Example: 
      #include <stdio.h>
int main() {
    char str[20];
    printf("Enter a string: ");
    fgets(str, 20, stdin);
    printf("You entered: %s", str);
    return 0;
}
  • Writing Strings:
    • printf("%s", str): Prints string.
    • puts(str): Prints string and adds newline.
    • Example: 
      #include <stdio.h>
int main() {
    char str[] = "Hello";
    puts(str); // Outputs: Hello\n
    return 0;
}

6.9 Null Character

The null character (\0) marks the end of a string in C.

  • Purpose: Indicates string termination for functions like printf, strlen.
  • Representation: ASCII value 0, written as \0.
  • Example: 
    #include <stdio.h>
int main() {
    char str[6] = {'H', 'e', 'l', 'l', 'o', '\0'};
    printf("%s\n", str); // Outputs: Hello
    return 0;
}
  • Note: String literals automatically include \0, but character arrays do not unless explicitly added.

6.10 Library Functions for String Handling

The C standard library (string.h) provides functions for string manipulation.

  • Common Functions:
    • strlen(str): Returns string length (excludes \0).
    • strcpy(dest, src): Copies src to dest, including \0.
    • strcat(dest, src): Appends src to dest.
    • strcmp(str1, str2): Compares strings (returns 0 if equal).
    • strchr(str, ch): Finds first occurrence of ch in str.
    • strstr(str, substr): Finds first occurrence of substr in str.
  • Example: 
    #include <stdio.h>
#include <string.h>
int main() {
    char str1[20] = "Hello";
    char str2[20] = "World";
    printf("Length of str1: %zu\n", strlen(str1)); // 5
    strcat(str1, " ");
    strcat(str1, str2);
    printf("Concatenated: %s\n", str1); // Hello World
    return 0;
}
  • Note: Ensure destination buffers are large enough to avoid buffer overflow.
