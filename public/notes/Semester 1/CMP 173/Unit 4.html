Unit 4: Cascading Style Sheets

4.1 Introduction to CSS

Cascading Style Sheets (CSS) is a stylesheet language used to control the presentation and layout of HTML documents. It separates content from design, enabling consistent styling across multiple pages.

  • Purpose: Enhances visual appeal and user experience.
  • Cascading: Styles are applied based on specificity and order of rules.
  • Versions: CSS3 is the latest, supporting animations, gradients, and more.

4.2 CSS Syntax

CSS syntax consists of selectors and declaration blocks. A declaration block contains properties and their values, enclosed in curly braces.

  • Selector: Targets HTML elements (e.g., h1, .class).
  • Property: Defines what to style (e.g., color).
  • Value: Specifies the style (e.g., blue).
h1 {
  color: blue;
  font-size: 24px;
}

4.3 Using CSS with HTML

CSS can be applied to HTML in three ways: inline, internal, and external.

  • Inline CSS: Uses the style attribute in HTML tags.
  • Internal CSS: Defined within a <style> tag in the <head>.
  • External CSS: Linked via a <link> tag to a separate .css file.
<!-- Inline CSS -->
<p style="color: red;">Inline text</p>

<!-- Internal CSS -->
<head>
  <style>
    p { color: blue; }
  </style>
</head>

<!-- External CSS -->
<link rel="stylesheet" href="styles.css">

4.4 CSS Selectors

Selectors target HTML elements to apply styles. They range from basic to complex for precise targeting.

  • Element Selector: Targets HTML tags (e.g., p).
  • Class Selector: Targets elements with a class (e.g., .class).
  • ID Selector: Targets a unique element (e.g., #id).
  • Pseudo-classes: Style elements based on state (e.g., :hover).
p { color: black; }
.my-class { font-size: 16px; }
#my-id { background: yellow; }
a:hover { color: red; }

4.5 CSS Comments

Comments in CSS are used to explain code or temporarily disable styles, improving maintainability.

  • Syntax: Enclosed in /* */.
  • Use: Add notes or debug by commenting out code.
/* This is a comment */
p {
  color: blue; /* Text color */
}

4.6 CSS Properties

CSS properties define the visual characteristics of elements. Below are key properties:

4.6.1 Backgrounds

Sets background color, image, or gradient for elements.

.box {
  background-color: #f0f0f0;
  background-image: url('image.jpg');
  background-size: cover;
}

4.6.2 Border, Margin

Defines borders and outer spacing around elements.

.box {
  border: 2px solid black;
  margin: 10px;
}

4.6.3 Padding

Adds inner spacing between content and borders.

.box {
  padding: 15px;
}

4.6.4 Height

Sets the height of an element.

.box {
  height: 200px;
}

4.6.5 Width

Sets the width of an element.

.box {
  width: 300px;
}

4.6.6 Color (Color Wheel)

Defines text or element color using names, hex, RGB, or HSL.

p {
  color: #ff0000; /* Red */
  color: rgb(0, 255, 0); /* Green */
  color: hsl(240, 100%, 50%); /* Blue */
}

4.7 Text

Controls text appearance, such as decoration, transformation, and spacing.

  • text-decoration: Adds underline, overline, or strikethrough.
  • text-transform: Changes case (e.g., uppercase).
  • letter-spacing: Adjusts space between characters.
p {
  text-decoration: underline;
  text-transform: uppercase;
  letter-spacing: 2px;
}

4.8 Font

Defines font properties like family, size, weight, and style.

  • font-family: Specifies the font (e.g., Arial).
  • font-size: Sets text size (e.g., 16px).
  • font-weight: Controls boldness (e.g., bold).
p {
  font-family: Arial, sans-serif;
  font-size: 16px;
  font-weight: bold;
}

4.9 Alignment

Controls the alignment of text or elements within their container.

  • text-align: Aligns text (e.g., center, justify).
  • vertical-align: Aligns inline elements vertically.
p {
  text-align: center;
}
img {
  vertical-align: middle;
}

4.10 Line Height

Adjusts the vertical spacing between lines of text for readability.

p {
  line-height: 1.8;
}

4.11 Box Model

The CSS box model represents the structure of elements, consisting of content, padding, border, and margin.

  • Content: The actual content (text, images).
  • Padding: Space inside the border.
  • Border: Surrounds padding and content.
  • Margin: Space outside the border.
.box {
  width: 200px;
  padding: 10px;
  border: 2px solid black;
  margin: 15px;
}

4.12 Working with Images

CSS styles images for size, positioning, and effects like borders or filters.

  • object-fit: Controls image scaling (e.g., cover, contain).
  • border-radius: Rounds image corners.
  • filter: Applies effects like blur or grayscale.
img {
  width: 100%;
  object-fit: cover;
  border-radius: 10px;
  filter: grayscale(50%);
}

4.13 Layout and Positioning

CSS controls element placement using positioning, flexbox, or grid layouts.

  • position: Static, relative, absolute, fixed, or sticky.
  • Flexbox: Flexible box layout for one-dimensional arrangements.
  • Grid: Two-dimensional layout system.
.container {
  display: flex;
  justify-content: space-between;
}
.box {
  position: relative;
  top: 10px;
}

4.14 Media Query

Media queries apply styles based on device characteristics, enabling responsive design.

  • Syntax: Uses @media to define conditions.
  • Breakpoints: Common for mobile, tablet, and desktop.
@media (max-width: 600px) {
  .container {
    flex-direction: column;
  }
}

4.15 CSS Website Layout

Combines CSS techniques to create structured, responsive website layouts.

  • Header, Main, Footer: Common layout structure.
  • Grid/Flexbox: Used for multi-column or responsive layouts.
  • Navigation: Styled menus for user navigation.
body {
  margin: 0;
}
.header {
  background: #333;
  color: white;
  padding: 10px;
}
.main {
  display: grid;
  grid-template-columns: 1fr 3fr;
}
@media (max-width: 768px) {
  .main {
    grid-template-columns: 1fr;
  }
}
