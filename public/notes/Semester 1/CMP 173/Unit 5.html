Unit 5: Advanced Topics on CSS

5.1 CSS Flexbox

Flexbox is a one-dimensional layout model that provides a flexible way to arrange items within a container, automatically adjusting their size and position.

  • display: flex: Enables flexbox layout for a container.
  • flex-direction: Defines the main axis (row or column).
  • justify-content: Aligns items along the main axis.
  • align-items: Aligns items along the cross axis.
.container {
  display: flex;
  flex-direction: row;
  justify-content: space-between;
  align-items: center;
}
.item {
  flex: 1;
}

5.2 CSS Grid

CSS Grid is a two-dimensional layout system that allows precise control over rows and columns, ideal for complex layouts.

  • display: grid: Enables grid layout for a container.
  • grid-template-columns/rows: Defines the structure of the grid.
  • gap: Sets spacing between grid cells.
  • grid-area: Assigns items to specific grid areas.
.grid-container {
  display: grid;
  grid-template-columns: 1fr 2fr 1fr;
  gap: 10px;
}
.grid-item {
  grid-column: 1 / 3;
}

5.3 CSS Transitions and Animations

Transitions and animations add dynamic effects to elements, enhancing user experience.

  • transition: Smoothly changes property values over time.
  • @keyframes: Defines animation sequences.
  • animation: Applies keyframes with timing and iteration.
.box {
  transition: background-color 0.3s ease;
}
.box:hover {
  background-color: blue;
}
@keyframes slide {
  from { transform: translateX(0); }
  to { transform: translateX(100px); }
}
.animated {
  animation: slide 2s infinite;
}

5.4 Responsive Web Design

Responsive Web Design (RWD) ensures websites adapt to different devices using flexible layouts, media queries, and relative units.

  • Media Queries: Apply styles based on screen size or device.
  • Relative Units: Use vw, vh, rem, em for scalability.
  • Flexible Images: Use max-width: 100% to prevent overflow.
img {
  max-width: 100%;
  height: auto;
}
@media (max-width: 768px) {
  .container {
    flex-direction: column;
  }
}

5.5 CSS Specificity and Inheritance

Specificity determines which CSS rule is applied when multiple rules target the same element. Inheritance passes styles from parent to child elements.

  • Specificity: Calculated by selector type (ID > Class > Element).
  • Inheritance: Some properties (e.g., color, font) are inherited.
  • !important: Overrides other rules but should be used sparingly.
#id { color: blue; } /* High specificity */
.class { color: red; } /* Medium specificity */
p { color: green; } /* Low specificity */
div { color: black; } /* Inherited by child elements */

5.6 CSS Units and Values

CSS supports various units for sizing and positioning, including absolute and relative units.

  • Absolute Units: px, pt, cm (fixed sizes).
  • Relative Units: rem, em, vw, vh, % (relative to parent or viewport).
  • Custom Properties: Use --variable for reusable values.
:root {
  --main-color: #3498db;
}
.box {
  width: 50vw;
  font-size: 1.5rem;
  background: var(--main-color);
}

5.7 CSS Preprocessors

CSS preprocessors like SASS and LESS extend CSS with variables, nesting, and mixins for better maintainability.

  • Variables: Store reusable values (e.g., colors).
  • Nesting: Write CSS hierarchically to match HTML structure.
  • Mixins: Reusable blocks of styles.
/* SASS Example */
$primary-color: #3498db;
.container {
  background: $primary-color;
  & .item {
    padding: 10px;
  }
}
@mixin border {
  border: 1px solid black;
}
.box {
  @include border;
}

5.8 CSS Best Practices and Optimization

Adopting best practices ensures efficient, maintainable, and performant CSS.

  • Minify CSS: Reduce file size by removing whitespace and comments.
  • Use Shorthand: Combine properties like margin or padding.
  • Avoid !important: Use specificity to manage rule priority.
  • Organize Code: Group related styles and use comments for clarity.
/* Optimized CSS */
.container {
  margin: 10px 20px; /* Shorthand */
  padding: 15px;
  background: #fff; /* Avoid inline styles */
}
/* Grouped navigation styles */
.nav {
  display: flex;
  gap: 10px;
}
