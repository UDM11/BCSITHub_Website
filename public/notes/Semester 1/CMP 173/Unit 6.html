Unit 6: Client Side Scripting with JavaScript

6.1 Introduction to JavaScript

JavaScript is a high-level, interpreted programming language used to make web pages interactive. It runs in the browser, enabling dynamic content and user interactions.

  • Purpose: Adds interactivity to websites.
  • Client-Side: Executes in the user’s browser, reducing server load.
  • ECMAScript: The standard JavaScript is based on, with ES6+ introducing modern features.

6.2 Using JS in HTML

JavaScript can be integrated into HTML using inline scripts, internal scripts, or external files.

  • Inline: Directly in HTML tags using event attributes.
  • Internal: Within a <script> tag in the <head> or <body>.
  • External: Linked via <script src="file.js"> for modularity.
<!-- Inline -->
<button onclick="alert('Clicked!')">Click Me</button>

<!-- Internal -->
<script>
  console.log('Hello, JavaScript!');
</script>

<!-- External -->
<script src="script.js"></script>

6.3 JavaScript Output

JavaScript provides multiple ways to display output, such as writing to the console, DOM, or alerts.

  • console.log(): Outputs to the browser’s console for debugging.
  • document.write(): Writes directly to the HTML document.
  • alert(): Displays a popup message.
  • innerHTML: Updates content of an HTML element.
console.log('Debug message');
alert('Welcome!');
document.write('Hello World');
document.getElementById('demo').innerHTML = 'Updated content';

6.4 JavaScript Comments

Comments explain code or disable execution, improving readability and maintenance.

  • Single-line: Use // for one line.
  • Multi-line: Use /* */ for multiple lines.
// This is a single-line comment
/* This is a
   multi-line comment */
let x = 5; // Initialize x

6.5 Variables and Data Types

Variables store data, and JavaScript supports various data types. Variables are declared using let, const, or var.

  • Primitive Types: Number, String, Boolean, Null, Undefined, Symbol.
  • Reference Types: Objects, Arrays, Functions.
  • let/const: Block-scoped; const prevents reassignment.
let name = 'John'; // String
const age = 30; // Number
let isStudent = true; // Boolean
let data = null; // Null
let obj = { key: 'value' }; // Object

6.6 Operators and Expressions

Operators perform operations on variables and values, forming expressions.

  • Arithmetic: +, -, *, /, %.
  • Comparison: ==, ===, !=, >, <.
  • Logical: &&, ||, !.
  • Assignment: =, +=, -=.
let a = 10;
let b = 5;
console.log(a + b); // 15
console.log(a === b); // false
console.log(a > b && b > 0); // true
a += 5; // a = 15

6.7 Control Flow and Conditionals

Conditionals control program flow based on logical conditions.

  • if/else: Executes code based on a condition.
  • switch: Handles multiple conditions with cases.
  • ternary operator: Concise condition ? expr1 : expr2.
let score = 85;
if (score >= 90) {
  console.log('A');
} else if (score >= 80) {
  console.log('B');
}
let day = 1;
switch (day) {
  case 1:
    console.log('Monday');
    break;
}
let result = score > 80 ? 'Pass' : 'Fail';

6.8 Loops

Loops execute code repeatedly based on conditions.

  • for: Iterates with initialization, condition, and increment.
  • while: Runs while a condition is true.
  • do-while: Executes at least once before checking the condition.
  • for...of: Iterates over iterable objects like arrays.
for (let i = 0; i < 5; i++) {
  console.log(i);
}
let j = 0;
while (j < 3) {
  console.log(j);
  j++;
}
let arr = [1, 2, 3];
for (let value of arr) {
  console.log(value);
}

6.9 Functions

Functions are reusable blocks of code that perform specific tasks.

  • Declaration: function name() {}.
  • Expression: const name = function() {}.
  • Arrow Functions: Concise syntax const name = () => {}.
  • Parameters/Return: Pass data and return results.
function greet(name) {
  return `Hello, ${name}`;
}
const add = (a, b) => a + b;
console.log(greet('John')); // Hello, John
console.log(add(2, 3)); // 5

6.10 Arrays

Arrays store ordered lists of data and provide methods for manipulation.

  • Creation: let arr = [1, 2, 3].
  • Methods: push(), pop(), map(), filter().
  • Access: Use index, e.g., arr[0].
let arr = [1, 2, 3];
arr.push(4); // [1, 2, 3, 4]
let doubled = arr.map(x => x * 2); // [2, 4, 6, 8]
console.log(arr[0]); // 1

6.11 Objects

Objects store key-value pairs, representing complex data structures.

  • Creation: let obj = { key: value }.
  • Access: Dot notation (obj.key) or bracket notation (obj['key']).
  • Methods: Functions as object properties.
let person = {
  name: 'John',
  age: 30,
  greet: function() {
    return `Hi, I'm ${this.name}`;
  }
};
console.log(person.name); // John
console.log(person['age']); // 30
console.log(person.greet()); // Hi, I'm John
