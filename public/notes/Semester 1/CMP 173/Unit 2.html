Unit 2: Hyper Text Markup Language

2.1 Introduction to HTML

HTML (Hyper Text Markup Language) is the standard language used to create and design web pages. It provides the structure for content on the web, allowing browsers to interpret and display text, images, and multimedia. HTML uses tags to define elements and their hierarchy within a document.

  • Purpose: Defines the structure and content of web pages.
  • Tags: Enclosed in angle brackets, e.g., <tag>, used to mark up content.
  • Versions: HTML5 is the latest standard, offering improved semantics and multimedia support.
<!DOCTYPE html>
<html>
<head>
  <title>My First Web Page</title>
</head>
<body>
  <h1>Welcome to HTML</h1>
  <p>This is a simple HTML page.</p>
</body>
</html>

2.2 Document Structure

An HTML document follows a specific structure to ensure proper rendering in browsers. The key components include the DOCTYPE declaration, <html>, <head>, and <body> tags.

  • <!DOCTYPE html>: Declares the document as HTML5.
  • <html>: The root element, containing all other elements.
  • <head>: Contains metadata, such as the page title and character set.
  • <body>: Contains the visible content of the page.
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Document Structure</title>
</head>
<body>
  <h1>My Page</h1>
  <p>This is the content.</p>
</body>
</html>

2.3 Text Formatting

HTML provides tags to format text, such as headings, paragraphs, and emphasis, to enhance readability and structure.

  • <h1> to <h6>: Define headings of varying importance.
  • <p>: Defines a paragraph.
  • <b>, <i>: Bold and italic text for visual emphasis.
  • <strong>, <em>: Semantic bold and italic for importance and emphasis.
<h1>Main Heading</h1>
<h2>Subheading</h2>
<p>This is a <strong>bold</strong> and <em>italic</em> text.</p>

2.4 Links and Navigation

Links allow users to navigate between pages or sections within a website, created using the <a> tag.

  • <a href="...">: Creates a hyperlink to another page or resource.
  • Attributes: target="_blank" opens links in a new tab; title provides a tooltip.
  • Navigation Menus: Use lists (<ul> or <nav>) for structured navigation.
<nav>
  <ul>
    <li><a href="home.html">Home</a></li>
    <li><a href="about.html" target="_blank">About</a></li>
  </ul>
</nav>

2.5 Hyperlink

Hyperlinks connect web pages and resources, enabling seamless navigation across the internet or within a site.

  • Absolute URLs: Full web addresses (e.g., https://example.com).
  • Relative URLs: Paths relative to the current page (e.g., page.html).
  • Anchor Links: Link to specific sections within a page using #id.
<a href="https://example.com">Visit Example</a>
<a href="about.html">About Page</a>
<a href="#section1">Go to Section 1</a>

2.6 Images and Multimedia

HTML supports embedding images and multimedia (audio, video) to enhance web content.

  • <img>: Embeds images with src and alt attributes.
  • <audio>: Embeds audio files with controls for playback.
  • <video>: Embeds videos with options for controls and autoplay.
<img src="image.jpg" alt="Description of image">
<audio controls>
  <source src="audio.mp3" type="audio/mpeg">
</audio>
<video controls>
  <source src="video.mp4" type="video/mp4">
</video>

2.7 Lists, Tables, Forms and Input

These elements organize and collect data effectively on web pages.

  • Lists: Ordered (<ol>) and unordered (<ul>) lists for structured content.
  • Tables: <table> organizes data in rows and columns.
  • Forms: <form> collects user input with elements like <input>, <textarea>.
<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>
<table border="1">
  <tr>
    <th>Header 1</th>
    <th>Header 2</th>
  </tr>
  <tr>
    <td>Data 1</td>
    <td>Data 2</td>
  </tr>
</table>
<form>
  <label for="name">Name:</label>
  <input type="text" id="name" name="name">
  <input type="submit" value="Submit">
</form>
