Unit 2: Boolean Building Blocks

2.1 Basic Laws of Boolean Algebra

Boolean algebra is a mathematical framework used to analyze and design digital circuits, manipulating binary variables (0 and 1) with operations like AND, OR, and NOT.

  • Identity Laws:
    • A + 0 = A (OR with 0 leaves A unchanged).
    • A · 1 = A (AND with 1 leaves A unchanged).
  • Null Laws:
    • A + 1 = 1 (OR with 1 is always 1).
    • A · 0 = 0 (AND with 0 is always 0).
  • Idempotent Laws:
    • A + A = A (OR with itself is A).
    • A · A = A (AND with itself is A).
  • Complement Laws:
    • A + A' = 1 (OR with complement is 1).
    • A · A' = 0 (AND with complement is 0).
  • Commutative Laws:
    • A + B = B + A (OR order doesn’t matter).
    • A · B = B · A (AND order doesn’t matter).
  • Associative Laws:
    • (A + B) + C = A + (B + C) (OR grouping).
    • (A · B) · C = A · (B · C) (AND grouping).
  • Distributive Laws:
    • A · (B + C) = A · B + A · C.
    • A + B · C = (A + B) · (A + C).
  • Absorption Laws:
    • A + A · B = A.
    • A · (A + B) = A.
  • Example: Simplify A + A · B. Using absorption law: A + A · B = A.
  • Applications: Simplify logic expressions and optimize digital circuits.

2.2 De Morgan's Theorems

De Morgan’s theorems provide rules to transform logical expressions involving complements, useful for circuit design and simplification.

  • First Theorem: (A + B)' = A' · B'. The complement of an OR operation is the AND of the complements.
  • Second Theorem: (A · B)' = A' + B'. The complement of an AND operation is the OR of the complements.
  • Example: Simplify (A + B)'. By De Morgan’s: (A + B)' = A' · B'. For (A · B)', (A · B)' = A' + B'.
  • Generalization: Extends to multiple variables, e.g., (A + B + C)' = A' · B' · C'.
  • Applications: Used to convert between AND/OR gates, simplify expressions, and design NAND/NOR-based circuits.

2.3 Logic Gates and Their Symbols

Logic gates are the building blocks of digital circuits, performing basic Boolean operations.

  • AND Gate: Outputs 1 if all inputs are 1. Symbol: A curved input with a flat output.
  • OR Gate: Outputs 1 if at least one input is 1. Symbol: A curved shape with a pointed output.
  • NOT Gate: Inverts the input (0 to 1, 1 to 0). Symbol: A triangle with a small circle at the output.
  • NAND Gate: AND followed by NOT. Outputs 0 if all inputs are 1. Symbol: AND gate with a circle at the output.
  • NOR Gate: OR followed by NOT. Outputs 1 if all inputs are 0. Symbol: OR gate with a circle at the output.
  • XOR Gate: Outputs 1 if exactly one input is 1. Symbol: OR gate with an extra curved line at the input.
  • XNOR Gate: Outputs 1 if inputs are the same. Symbol: XOR gate with a circle at the output.
  • Applications: Used in processors, memory units, and control circuits.

2.5 Universal Gates

Universal gates can implement any Boolean function, reducing the need for multiple gate types.

  • NAND Gate: Universal because any gate (AND, OR, NOT) can be constructed using only NAND gates.
    • NOT: A NAND A = (A · A)' = A'.
    • AND: (A NAND B)' = ((A · B)')' = A · B.
    • OR: (A' NAND B') = (A' · B')' = A + B (using De Morgan’s).
    • Example: Implement A + B using NAND: (A NAND A) NAND (B NAND B) = A' NAND B' = A + B.
  • NOR Gate: Universal because any gate can be constructed using only NOR gates.
    • NOT: A NOR A = (A + A)' = A'.
    • OR: (A NOR B)' = ((A + B)')' = A + B.
    • AND: (A' NOR B') = (A' + B')' = A · B (using De Morgan’s).
    • Example: Implement A · B using NOR: (A NOR A) NOR (B NOR B) = A' NOR B' = A · B.
  • Applications: NAND and NOR gates simplify circuit design, reduce manufacturing costs, and are used in integrated circuits.
