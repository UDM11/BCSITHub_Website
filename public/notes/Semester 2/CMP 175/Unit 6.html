Unit 6: Streams in Java

6.1 Introduction to Stream, Byte Stream, and Character Stream

A stream in Java is a sequence of data elements made available over time, used for input/output (I/O) operations such as reading from or writing to files, networks, or other sources. Java provides a robust I/O framework through the java.io package.

  • Byte Streams: Handle raw binary data (bytes). Used for reading/writing data like images, audio, or raw files. Classes include InputStream and OutputStream.
  • Character Streams: Handle text data (Unicode characters). Used for reading/writing text files or strings. Classes include Reader and Writer.

Example (Byte Stream vs. Character Stream):

import java.io.*;

public class StreamExample {
    public static void main(String[] args) throws IOException {
        // Byte Stream
        FileInputStream fis = new FileInputStream("bytefile.txt");
        int data = fis.read(); // Reads a byte
        fis.close();

        // Character Stream
        FileReader fr = new FileReader("charfile.txt");
        int charData = fr.read(); // Reads a character
        fr.close();
    }
}

6.2 I/O Class Hierarchy

The java.io package provides a hierarchy of classes for handling I/O operations. The main base classes are:

  • InputStream: Abstract class for byte input streams (e.g., FileInputStream, ObjectInputStream).
  • OutputStream: Abstract class for byte output streams (e.g., FileOutputStream, ObjectOutputStream).
  • Reader: Abstract class for character input streams (e.g., FileReader, BufferedReader).
  • Writer: Abstract class for character output streams (e.g., FileWriter, BufferedWriter).

Key characteristics:

  • Byte stream classes handle raw binary data, while character stream classes handle text with proper encoding.
  • Buffered streams (e.g., BufferedInputStream, BufferedReader) improve performance by reducing direct access to underlying resources.

6.3 Manipulating File

Java provides several classes for file manipulation under the java.io package. Below are key classes for handling file I/O:

File Input Stream

FileInputStream reads raw bytes from a file. Used for binary data or non-text files.

Example:

import java.io.*;

public class FileInputStreamExample {
    public static void main(String[] args) throws IOException {
        FileInputStream fis = new FileInputStream("input.txt");
        int data;
        while ((data = fis.read()) != -1) { // Read byte by byte
            System.out.print((char) data);
        }
        fis.close();
    }
}

File Output Stream

FileOutputStream writes raw bytes to a file. Used for binary data or non-text files.

Example:

import java.io.*;

public class FileOutputStreamExample {
    public static void main(String[] args) throws IOException {
        FileOutputStream fos = new FileOutputStream("output.txt");
        String text = "Hello, Java!";
        fos.write(text.getBytes()); // Write bytes
        fos.close();
    }
}

Data Input Stream

DataInputStream reads primitive Java data types (e.g., int, double) from an underlying input stream in a machine-independent way.

Example:

import java.io.*;

public class DataInputStreamExample {
    public static void main(String[] args) throws IOException {
        FileInputStream fis = new FileInputStream("data.bin");
        DataInputStream dis = new DataInputStream(fis);
        int number = dis.readInt(); // Read an integer
        System.out.println("Number: " + number);
        dis.close();
        fis.close();
    }
}

Data Output Stream

DataOutputStream writes primitive Java data types to an output stream in a machine-independent way.

Example:

import java.io.*;

public class DataOutputStreamExample {
    public static void main(String[] args) throws IOException {
        FileOutputStream fos = new FileOutputStream("data.bin");
        DataOutputStream dos = new DataOutputStream(fos);
        dos.writeInt(100); // Write an integer
        dos.close();
        fos.close();
    }
}

Object Input Stream

ObjectInputStream deserializes Java objects and primitive data from an input stream. The objects must implement the Serializable interface.

Example:

import java.io.*;

public class ObjectInputStreamExample {
    public static void main(String[] args) throws IOException, ClassNotFoundException {
        FileInputStream fis = new FileInputStream("object.dat");
        ObjectInputStream ois = new ObjectInputStream(fis);
        Object obj = ois.readObject(); // Read an object
        System.out.println("Object: " + obj);
        ois.close();
        fis.close();
    }
}

Object Output Stream

ObjectOutputStream serializes Java objects and primitive data to an output stream. The objects must implement the Serializable interface.

Example:

import java.io.*;

public class ObjectOutputStreamExample {
    public static void main(String[] args) throws IOException {
        FileOutputStream fos = new FileOutputStream("object.dat");
        ObjectOutputStream oos = new ObjectOutputStream(fos);
        oos.writeObject("Hello, Java!"); // Write an object
        oos.close();
        fos.close();
    }
}
