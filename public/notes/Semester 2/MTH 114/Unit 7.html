Unit 7: Fourier Series and Integrals

7.1 Introduction

Fourier series and integrals are mathematical tools used to represent functions as sums or integrals of sine and cosine functions. They are fundamental in analyzing periodic phenomena and solving differential equations in physics, engineering, and signal processing.

  • Purpose: Decompose complex functions into simpler trigonometric components to study their behavior. Example: Representing a sound wave as a sum of pure tones.
  • Fourier Series: Expresses a periodic function as a sum of sines and cosines. Example: A square wave can be approximated by a series of sine terms.
  • Fourier Integrals: Extend Fourier series to non-periodic functions, representing them as integrals over a continuous range of frequencies.
  • Applications: Used in signal processing (e.g., audio analysis), heat transfer, vibration analysis, and image compression.

7.2 Even and Odd Function

Even and odd functions have symmetry properties that simplify Fourier series representations.

  • Even Function: A function f(x) is even if f(-x) = f(x) for all x. Graphically, it is symmetric about the y-axis. Example: f(x) = cos x, f(x) = x².
  • Odd Function: A function f(x) is odd if f(-x) = -f(x). Graphically, it is symmetric about the origin. Example: f(x) = sin x, f(x) = x³.
  • Properties:
    • Integral of an odd function over a symmetric interval [-a, a] is zero: ∫-aa f(x) dx = 0.
    • Integral of an even function: ∫-aa f(x) dx = 2 ∫0a f(x) dx.
    Example: For f(x) = x (odd), ∫-11 x dx = [x²/2]-11 = 1/2 - 1/2 = 0.
  • Application in Fourier Series: Even functions yield only cosine terms; odd functions yield only sine terms.

7.3 Periodic Function

A periodic function repeats its values at regular intervals, which is essential for Fourier series.

  • Definition: A function f(x) is periodic with period T if f(x + T) = f(x) for all x. The smallest such T is the fundamental period. Example: f(x) = sin x has period 2π.
  • Examples: cos x (period 2π), sin(2x) (period π), square wave (period depends on definition).
  • Importance: Fourier series represent periodic functions as sums of sines and cosines with frequencies that are multiples of the fundamental frequency 1/T.
  • Applications: Model periodic phenomena like electrical signals, seasonal patterns, or mechanical vibrations.

7.4 Fourier Series and Fourier Coefficients (Without Proof)

A Fourier series represents a periodic function as an infinite sum of sines and cosines.

  • Definition: For a function f(x) with period 2L, the Fourier series is:

    f(x) = a₀/2 + ∑n=1∞ [aₙ cos(nπx/L) + bₙ sin(nπx/L)]

    where a₀, aₙ, bₙ are Fourier coefficients.
  • Fourier Coefficients:
    • a₀ = (1/L) ∫-LL f(x) dx
    • aₙ = (1/L) ∫-LL f(x) cos(nπx/L) dx, for n = 1, 2, ...
    • bₙ = (1/L) ∫-LL f(x) sin(nπx/L) dx, for n = 1, 2, ...
    Example: For f(x) = x on [-π, π], compute a₀ = (1/π) ∫-ππ x dx = 0 (odd function), aₙ = 0 (since f(x) cos(nπx/π) is odd), bₙ = (2/π) ∫0π x sin(nx) dx = (-2/n)(-1)^n. Thus, f(x) = ∑n=1∞ [(-2/n)(-1)^n sin(nx)].
  • Convergence: The series converges to f(x) at points of continuity, and to the average of the limits at discontinuities, assuming f(x) is piecewise smooth.

7.5 Fourier Sine and Cosine Series

Fourier sine and cosine series are used for odd or even extensions of functions, simplifying the Fourier series for specific symmetries.

  • Fourier Cosine Series: For an even function on [0, L], extended to [-L, L], the series contains only cosine terms:

    f(x) = a₀/2 + ∑n=1∞ aₙ cos(nπx/L)

    where a₀ = (2/L) ∫0L f(x) dx, aₙ = (2/L) ∫0L f(x) cos(nπx/L) dx. Example: For f(x) = x² on [0, π], a₀ = (2/π) ∫0π x² dx = 2π²/3, aₙ = (2/π) ∫0π x² cos(nx) dx.
  • Fourier Sine Series: For an odd function on [0, L], extended to [-L, L], the series contains only sine terms:

    f(x) = ∑n=1∞ bₙ sin(nπx/L)

    where bₙ = (2/L) ∫0L f(x) sin(nπx/L) dx. Example: For f(x) = x on [0, π], bₙ = (2/π) ∫0π x sin(nx) dx = (-2/n)(-1)^n.
  • Applications: Used in solving partial differential equations (e.g., heat equation) with specific boundary conditions.

7.6 Fourier Integral

The Fourier integral extends Fourier series to non-periodic functions, representing them as integrals over a continuous range of frequencies.

  • Definition: For a function f(x) defined on (-∞, ∞), the Fourier integral is:

    f(x) = ∫0∞ [A(ω) cos(ωx) + B(ω) sin(ωx)] dω

    where A(ω) = (1/π) ∫-∞∞ f(t) cos(ωt) dt, B(ω) = (1/π) ∫-∞∞ f(t) sin(ωt) dt.
  • Conditions: f(x) must be piecewise continuous, have a finite number of discontinuities, and be absolutely integrable (∫-∞∞ |f(x)| dx < ∞).
  • Example: For f(x) = e^(-|x|), compute A(ω) = (2/π) ∫0∞ e^(-t) cos(ωt) dt = (2/π)(1/(1 + ω²)), B(ω) = 0 (since f(x) is even). Thus, f(x) = (2/π) ∫0∞ [1/(1 + ω²)] cos(ωx) dω.
  • Applications: Used in signal processing, heat conduction, and wave analysis for non-periodic functions.

7.7 Fourier Sine and Cosine Integral

Fourier sine and cosine integrals represent non-periodic functions using only sine or cosine terms, often for functions defined on [0, ∞).

  • Fourier Cosine Integral: For an even extension of f(x) on [0, ∞):

    f(x) = (2/π) ∫0∞ [∫0∞ f(t) cos(ωt) dt] cos(ωx) dω

    Example: For f(x) = 1 on [0, 1], 0 elsewhere, compute ∫0∞ f(t) cos(ωt) dt = ∫01 cos(ωt) dt = sin(ω)/ω. Thus, f(x) ≈ (2/π) ∫0∞ [sin(ω)/ω] cos(ωx) dω.
  • Fourier Sine Integral: For an odd extension of f(x) on [0, ∞):

    f(x) = (2/π) ∫0∞ [∫0∞ f(t) sin(ωt) dt] sin(ωx) dω

    Example: For f(x) = x on [0, 1], 0 elsewhere, compute ∫0∞ f(t) sin(ωt) dt = ∫01 t sin(ωt) dt, which requires integration by parts.
  • Applications: Solve boundary value problems (e.g., heat equation with Dirichlet or Neumann conditions) and analyze signals with specific symmetries.
