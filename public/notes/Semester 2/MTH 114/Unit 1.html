Unit 1: Complex Numbers

1.1 Definition of a Complex Number, Integral Powers of i

A complex number is a number of the form z = a + bi, where a and b are real numbers, and i is the imaginary unit with the property i² = -1. Here, a is the real part, and b is the imaginary part.

  • Definition: A complex number combines a real number and an imaginary number. Example: 3 + 4i has real part 3 and imaginary part 4.
  • Integral Powers of i: The imaginary unit i follows a cyclic pattern when raised to integer powers:
    • i¹ = i
    • i² = -1
    • i³ = i² · i = -i
    • i⁴ = (i²)² = (-1)² = 1
    • i⁵ = i⁴ · i = 1 · i = i, and the cycle repeats every four powers.
    Example: To find i^7, note that 7 = 4 + 3, so i^7 = i⁴ · i³ = 1 · (-i) = -i.
  • General Rule: For any integer n, compute i^n by finding n mod 4: i^0 = 1, i^1 = i, i^2 = -1, i^3 = -i.

1.2 Algebra of Complex Numbers

Complex numbers can be added, subtracted, multiplied, and divided using standard algebraic operations, treating i as a variable with i² = -1.

  • Sum: Add real and imaginary parts separately. For z₁ = a + bi and z₂ = c + di, z₁ + z₂ = (a + c) + (b + d)i. Example: (3 + 4i) + (2 - 5i) = (3 + 2) + (4 - 5)i = 5 - i.
  • Difference: Subtract real and imaginary parts. z₁ - z₂ = (a - c) + (b - d)i. Example: (3 + 4i) - (2 - 5i) = (3 - 2) + (4 - (-5))i = 1 + 9i.
  • Multiplication: Use the distributive property and simplify using i² = -1. z₁ · z₂ = (a + bi)(c + di) = ac + adi + bci + bdi² = (ac - bd) + (ad + bc)i. Example: (3 + 4i)(2 - 5i) = 6 - 15i + 8i - 20i² = 6 + 20 - 7i = 26 - 7i.
  • Division: Multiply numerator and denominator by the conjugate of the denominator to make the denominator real. For z₁ / z₂ = (a + bi) / (c + di), multiply by (c - di) / (c - di). Example: (3 + 4i) / (2 + i) = [(3 + 4i)(2 - i)] / [(2 + i)(2 - i)] = (6 - 3i + 8i - 4i²) / (4 - i²) = (10 + 5i) / 5 = 2 + i.

1.3 Properties of Complex Numbers, Conjugate of a Complex Number

Complex numbers have specific properties that govern their behavior, and the conjugate is a key concept.

  • Properties (Without Proof):
    • Commutative: z₁ + z₂ = z₂ + z₁ and z₁ · z₂ = z₂ · z₁.
    • Associative: (z₁ + z₂) + z₃ = z₁ + (z₂ + z₃) and (z₁ · z₂) · z₃ = z₁ · (z₂ · z₃).
    • Distributive: z₁ · (z₂ + z₃) = z₁ · z₂ + z₁ · z₃.
    • Additive Identity: z + 0 = z, where 0 = 0 + 0i.
    • Multiplicative Identity: z · 1 = z, where 1 = 1 + 0i.
  • Conjugate: For z = a + bi, the conjugate is z̅ = a - bi. Example: Conjugate of 3 + 4i is 3 - 4i.
  • Properties of Conjugate:
    • z + z̅ = 2a (real number). Example: (3 + 4i) + (3 - 4i) = 6.
    • z - z̅ = 2bi (purely imaginary). Example: (3 + 4i) - (3 - 4i) = 8i.
    • z · z̅ = a² + b² (real number). Example: (3 + 4i)(3 - 4i) = 9 + 16 = 25.
    • (z̅)̅ = z. Example: Conjugate of 3 - 4i is 3 + 4i.

1.4 Modulus of a Complex Number and Argand’s Diagram

The modulus measures the magnitude of a complex number, and Argand’s diagram represents it geometrically.

  • Modulus: For z = a + bi, the modulus is |z| = √(a² + b²), representing the distance from the origin in the complex plane. Example: For 3 + 4i, |z| = √(3² + 4²) = √25 = 5.
  • Properties of Modulus (Without Proof):
    • |z| ≥ 0, and |z| = 0 if and only if z = 0.
    • |z₁ · z₂| = |z₁| · |z₂|. Example: For z₁ = 3 + 4i and z₂ = 1 + i, verify using moduli.
    • |z₁ / z₂| = |z₁| / |z₂| (if z₂ ≠ 0).
    • |z̅| = |z|. Example: Modulus of 3 - 4i is also 5.
  • Argand’s Diagram: Represents a complex number z = a + bi as a point (a, b) in a 2D plane, with the x-axis as the real axis and the y-axis as the imaginary axis. Example: 3 + 4i is plotted at (3, 4).

1.5 Polar Representation of a Complex Number

Polar form expresses a complex number using its modulus and argument (angle).

  • Definition: For z = a + bi, polar form is z = r(cosθ + i sinθ), where r = |z| = √(a² + b²) is the modulus, and θ = tan⁻¹(b/a) is the argument (adjusted for quadrant). Example: For 3 + 4i, r = 5, θ ≈ tan⁻¹(4/3) ≈ 53.13°, so z = 5(cos 53.13° + i sin 53.13°).
  • Argument: The angle θ is measured from the positive real axis, with -π < θ ≤ π (principal value). Adjust θ based on the quadrant of (a, b).
  • Advantages: Simplifies multiplication, division, and powers of complex numbers. Example: Multiplication involves multiplying moduli and adding arguments.

1.6 Square Roots of a Complex Number (Cartesian Form)

The square root of a complex number z = a + bi yields two complex numbers whose square equals z.

  • Method: Let the square root be w = x + yi. Then w² = z, so (x + yi)² = x² - y² + 2xyi = a + bi. Equate real and imaginary parts:
    • x² - y² = a
    • 2xy = b
    • Solve with |w| = √|z|, where x² + y² = √(a² + b²).
    Example: For z = 3 + 4i, solve for w. Let x² + y² = √(3² + 4²) = 5. From 2xy = 4, y = 2/x. Substitute into x² - (2/x)² = 3, yielding x⁴ - 3x² - 4 = 0. Solve to get x² = 4 or x² = -1 (discard negative). Thus, x = ±2, y = ±1. Roots are 2 + i and -2 - i.
  • Verification: Check: (2 + i)² = 4 + 4i + i² = 3 + 4i. Similarly for -2 - i.

1.7 De Moivre’s Theorem and Its Application

De Moivre’s theorem provides a method to compute powers and roots of complex numbers in polar form.

  • Statement: For a complex number z = r(cosθ + i sinθ) and integer n, z^n = r^n (cos(nθ) + i sin(nθ)).
  • Applications:
    • Powers: Simplify high powers of complex numbers. Example: For z = 1 + i, polar form is √2 (cos 45° + i sin 45°). Then z³ = (√2)³ (cos(3·45°) + i sin(3·45°)) = 2√2 (cos 135° + i sin 135°) = 2√2 (-1/√2 + i/√2) = -2 + 2i.
    • Roots: Find n-th roots using z^(1/n) = r^(1/n) [cos((θ + 2kπ)/n) + i sin((θ + 2kπ)/n)] for k = 0, 1, ..., n-1. Example: Cube roots of 1 = 1 + 0i (r = 1, θ = 0) are cos(2kπ/3) + i sin(2kπ/3) for k = 0, 1, 2, yielding 1, -1/2 + i√3/2, -1/2 - i√3/2.
  • Best Practices: Convert to polar form for easier computation, especially for high powers or roots.
