Unit 5: Ordinary Differential Equation

5.1 Introduction

An ordinary differential equation (ODE) is an equation involving a function of one independent variable and its derivatives. ODEs describe relationships between a function and its rates of change, widely used to model phenomena in physics, engineering, biology, and economics.

  • Definition: An ODE is of the form F(x, y, y', y'', ..., y^(n)) = 0, where y = f(x) is the unknown function, and y', y'', ..., y^(n) are its derivatives. Example: y' + 2y = x is an ODE.
  • Applications: Model population growth (e.g., dy/dt = ky), motion (e.g., Newton’s second law), or electrical circuits (e.g., L di/dt + Ri = V).
  • Types: Classified by order, degree, and linearity (linear or nonlinear). Solutions can be explicit, implicit, or numerical.

5.2 Order and Degree of Differential Equation

The order and degree characterize the structure of a differential equation, indicating its complexity and the methods needed for solution.

  • Order: The highest derivative in the equation. Example: y'' + 3y' + 2y = 0 is second-order (highest derivative is y''); y' = x² is first-order.
  • Degree: The power of the highest-order derivative when the equation is polynomial in derivatives. Example: (y'')² + y' + y = 0 has degree 2 (due to (y'')²); y'' + y = 0 has degree 1.
  • Determination: Rewrite the equation to isolate derivatives. Example: √(y') + y = x has order 1, degree 1/2 (after squaring to eliminate the square root). If the equation is not polynomial in derivatives, degree may be undefined.
  • Applications: Order determines the number of initial conditions needed; degree affects solution techniques.

5.3 Solution of the First Order and First Degree Differential Equation

First-order, first-degree ODEs have the form dy/dx = f(x, y). Solutions can be general (family of solutions) or particular (satisfying specific conditions).

  • General Solution: Includes an arbitrary constant. Example: For dy/dx = x, integrate: y = ∫ x dx = x²/2 + C.
  • Particular Solution: Obtained by applying an initial condition. Example: For y(0) = 1, y = x²/2 + 1.
  • Methods:
    • Direct Integration: If dy/dx = f(x), then y = ∫ f(x) dx + C. Example: dy/dx = 2x, y = x² + C.
    • Other Methods: Separation of variables, integrating factors (for linear equations), or exact equations (covered in 5.4).
  • Example: Solve dy/dx = -y/x. This is separable (see 5.4), yielding y = C/x.
  • Applications: Model exponential growth/decay, cooling processes, or simple motion.

5.4 Variable Separation, Homogeneous, Linear Differential Equation

Specific types of first-order ODEs can be solved using tailored methods, each suited to the equation’s structure.

  • Variable Separation: For dy/dx = f(x)g(y), rewrite as dy/g(y) = f(x) dx, then integrate both sides. Example: Solve dy/dx = xy. Rewrite: dy/y = x dx. Integrate: ln |y| = x²/2 + C, so y = ±e^(C) e^(x²/2) = A e^(x²/2).
  • Homogeneous Equations: An equation dy/dx = f(x, y) where f(tx, ty) = f(x, y) (same degree in x and y). Use substitution y = vx, so dy/dx = v + x dv/dx. Example: Solve dy/dx = (x + y)/x. Rewrite as dy/dx = 1 + y/x. Let y = vx, then v + x dv/dx = 1 + v, so x dv/dx = 1. Integrate: v = ln |x| + C, so y = x ln |x| + Cx.
  • Linear Differential Equations: Form dy/dx + P(x)y = Q(x). Use integrating factor e^(∫ P(x) dx). Multiply through by the integrating factor, then integrate. Example: Solve dy/dx + 2y/x = x. Integrating factor: e^(∫ 2/x dx) = e^(2 ln x) = x². Multiply: x² dy/dx + 2xy = x³, or d/dx (x²y) = x³. Integrate: x²y = x⁴/4 + C, so y = x²/4 + C/x².
  • Applications: Used in modeling population dynamics, fluid flow, and electrical circuits.

5.5 Second-Order Linear Differential Equation with Constant Coefficients

Second-order linear ODEs with constant coefficients have the form ay'' + by' + cy = f(x), where a, b, c are constants.

  • Homogeneous Case (f(x) = 0): Solve ay'' + by' + cy = 0. Use characteristic equation ar² + br + c = 0.
    • Real Distinct Roots: If roots r₁, r₂ are distinct, solution is y = C₁ e^(r₁x) + C₂ e^(r₂x). Example: y'' - 3y' + 2y = 0, characteristic equation r² - 3r + 2 = 0 gives r = 1, 2, so y = C₁ e^x + C₂ e^(2x).
    • Real Repeated Roots: If r₁ = r₂ = r, solution is y = (C₁ + C₂x) e^(rx). Example: y'' - 2y' + y = 0, r² - 2r + 1 = 0, r = 1, so y = (C₁ + C₂x) e^x.
    • Complex Roots: If r = α ± βi, solution is y = e^(αx) (C₁ cos(βx) + C₂ sin(βx)). Example: y'' + y = 0, r² + 1 = 0, r = ±i, so y = C₁ cos x + C₂ sin x.
  • Non-Homogeneous Case: Solve ay'' + by' + cy = f(x). Solution is y = y_h + y_p, where y_h is the homogeneous solution, and y_p is a particular solution found via:
    • Undetermined Coefficients: Guess y_p based on f(x). Example: For y'' + y = x, y_h = C₁ cos x + C₂ sin x. Guess y_p = ax + b, solve to get y_p = x, so y = C₁ cos x + C₂ sin x + x.
    • Variation of Parameters: Use y_p = u₁(x)y₁ + u₂(x)y₂, where y₁, y₂ are homogeneous solutions.
  • Applications: Model damped oscillations, electrical circuits, and mechanical systems.

5.6 Initial and Boundary Value Problems

Initial and boundary value problems specify conditions to find particular solutions to ODEs.

  • Initial Value Problems (IVPs): Specify conditions at a single point, typically x = x₀. For a second-order ODE, provide y(x₀) = y₀ and y'(x₀) = y₁. Example: Solve y'' + y = 0, y(0) = 1, y'(0) = 0. General solution: y = C₁ cos x + C₂ sin x, y' = -C₁ sin x + C₂ cos x. Apply conditions: C₁ = 1, C₂ = 0, so y = cos x.
  • Boundary Value Problems (BVPs): Specify conditions at different points, e.g., y(a) = y₁, y(b) = y₂. Example: Solve y'' + y = 0, y(0) = 0, y(π) = 0. General solution: y = C₁ cos x + C₂ sin x. Apply y(0) = C₁ = 0, y(π) = C₂ sin π = 0, so y = C₂ sin x (includes trivial solution y = 0).
  • Uniqueness: IVPs typically have unique solutions for linear ODEs; BVPs may have multiple or no solutions depending on conditions.
  • Applications: IVPs model dynamic systems (e.g., projectile motion); BVPs model steady-state systems (e.g., beam deflection).
