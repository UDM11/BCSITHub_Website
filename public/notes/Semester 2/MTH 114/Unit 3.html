Unit 3: Application of Antiderivative

3.1 Definite Integral

The definite integral represents the accumulation of a quantity over an interval, often interpreted as the area under a curve or the total change of a function. For a function f(x) continuous on [a, b], the definite integral is defined as:

∫ab f(x) dx = F(b) - F(a), where F(x) is the antiderivative of f(x) (i.e., F'(x) = f(x)).

  • Purpose: Computes areas, total quantities, or net change. Example: ∫02 x² dx represents the area under y = x² from x = 0 to x = 2.
  • Calculation: Find the antiderivative F(x), then evaluate F(b) - F(a). Example: For ∫02 x² dx, F(x) = x³/3, so F(2) - F(0) = (8/3) - 0 = 8/3.
  • Geometric Interpretation: The definite integral gives the signed area between the curve and the x-axis, with areas below the axis counted as negative.
  • Applications: Used in physics (work, displacement), economics (consumer surplus), and engineering (volume of solids).

3.2 Properties of the Definite Integral

The definite integral has properties that simplify calculations and aid in understanding its behavior.

  • Linearity: ∫ab [k f(x) + g(x)] dx = k ∫ab f(x) dx + ∫ab g(x) dx, where k is a constant. Example: ∫01 (2x + 3) dx = 2 ∫01 x dx + ∫01 3 dx.
  • Interval Additivity: If a < c < b, then ∫ab f(x) dx = ∫ac f(x) dx + ∫cb f(x) dx. Example: Split ∫04 x dx at x = 2.
  • Reversal of Limits: ∫ab f(x) dx = - ∫ba f(x) dx. Example: ∫20 x dx = - ∫02 x dx.
  • Zero Interval: ∫aa f(x) dx = 0. Example: No area is accumulated over a point.
  • Constant Function: ∫ab k dx = k(b - a). Example: ∫13 5 dx = 5(3 - 1) = 10.
  • Non-negativity: If f(x) ≥ 0 on [a, b], then ∫ab f(x) dx ≥ 0. Example: ∫01 x² dx ≥ 0 since x² ≥ 0.

Use: These properties simplify complex integrals and aid in problem-solving.

3.3 Improper Integral

An improper integral is a definite integral where either the interval is infinite or the integrand is unbounded (e.g., has a discontinuity) within the interval.

  • Types:
    • Infinite Limits: ∫a∞ f(x) dx = limt→∞ ∫at f(x) dx. Converges if the limit exists and is finite. Example: ∫1∞ 1/x² dx = limt→∞ [-1/x]1t = limt→∞ (-1/t + 1) = 1 (converges).
    • Unbounded Integrand: If f(x) is unbounded at c ∈ [a, b], then ∫ab f(x) dx = limt→c⁻ ∫at f(x) dx + lims→c⁺ ∫sb f(x) dx. Example: ∫01 1/√x dx = limt→0⁺ [2√x]t1 = limt→0⁺ (2 - 2√t) = 2 (converges).
  • Convergence: An improper integral converges if the limit exists; otherwise, it diverges. Example: ∫1∞ 1/x dx = limt→∞ [ln x]1t = limt→∞ ln t = ∞ (diverges).
  • Applications: Used in probability (e.g., normal distribution) and physics (e.g., work over infinite distances).

3.4 Quadrature [y = f(x)]

Quadrature refers to the process of finding the area under a curve y = f(x) over an interval [a, b] using the definite integral.

  • Definition: The area under y = f(x) from x = a to x = b is ∫ab f(x) dx, assuming f(x) ≥ 0. If f(x) takes negative values, the integral gives the net signed area.
  • Example: Find the area under y = x² from x = 0 to x = 2. Compute ∫02 x² dx = [x³/3]02 = 8/3.
  • Area Between Curves: For two functions f(x) and g(x), the area between them from x = a to x = b is ∫ab |f(x) - g(x)| dx. Example: Area between y = x and y = x² from x = 0 to x = 1 is ∫01 (x - x²) dx = [x²/2 - x³/3]01 = 1/2 - 1/3 = 1/6.
  • Applications: Used in economics (e.g., total revenue), physics (e.g., work done), and geometry (e.g., area calculations).

Rectification is the process of finding the arc length of a curve y = f(x) over an interval [a, b].

  • Formula: The arc length L of y = f(x) from x = a to x = b, where f(x) is differentiable, is L = ∫ab √(1 + [f'(x)]²) dx.
  • Example: Find the arc length of y = x² from x = 0 to x = 1. Compute f'(x) = 2x, so L = ∫01 √(1 + 4x²) dx. Use substitution: let u = 2x, du = 2 dx, dx = du/2, and adjust limits (x = 0 → u = 0, x = 1 → u = 2). Then L = (1/2) ∫02 √(1 + u²) du. This integral requires trigonometric substitution or numerical methods, yielding approximately 1.479.
  • Applications: Used in engineering (e.g., cable length), physics (e.g., particle paths), and computer graphics (e.g., curve modeling).

3.6 Beta and Gamma Function

The Beta and Gamma functions are special functions used to evaluate integrals, particularly in probability, physics, and statistics.

  • Gamma Function:
    • Definition: For x > 0, Γ(x) = ∫0∞ t^(x-1) e^(-t) dt. It generalizes the factorial: Γ(n) = (n-1)! for positive integers n.
    • Properties: Γ(x+1) = x Γ(x), Γ(1) = 1, Γ(1/2) = √π. Example: Γ(3) = 2! = 2.
    • Example: Evaluate ∫0∞ x² e^(-x) dx. This is Γ(3) = 2! = 2.
    • Applications: Used in probability (e.g., gamma distribution) and solving integrals in physics.
  • Beta Function:
    • Definition: For m, n > 0, B(m, n) = ∫01 x^(m-1) (1-x)^(n-1) dx. It is related to the Gamma function: B(m, n) = Γ(m)Γ(n) / Γ(m+n).
    • Example: Evaluate B(2, 3) = ∫01 x (1-x)² dx. Using the Gamma relation, B(2, 3) = Γ(2)Γ(3) / Γ(5) = 1! · 2! / 4! = 2/24 = 1/12. Alternatively, compute directly: ∫01 x (1 - 2x + x²) dx = [x²/2 - 2x³/3 + x⁴/4]01 = 1/2 - 2/3 + 1/4 = 1/12.
    • Applications: Used in probability (e.g., beta distribution), statistics, and solving integrals in engineering.
