Unit 2: Infinite Sequence and Series

2.1 Introduction

An infinite sequence is an ordered list of numbers that continues indefinitely, denoted as {aₙ}, where aₙ is the n-th term. An infinite series is the sum of the terms of an infinite sequence, written as ∑ aₙ (from n=1 to infinity). These concepts are fundamental in understanding convergence and divergence in mathematical analysis.

  • Sequence: A function from the natural numbers to real numbers. Example: The sequence {1/n} is 1, 1/2, 1/3, 1/4, ....
  • Series: The sum of a sequence’s terms. Example: The series ∑ 1/n is 1 + 1/2 + 1/3 + 1/4 + ....
  • Convergence: A sequence converges if its terms approach a finite limit as n approaches infinity; a series converges if the sum of its terms approaches a finite value. Example: The sequence {1/n} converges to 0; the geometric series ∑ (1/2)ⁿ converges to 1.
  • Divergence: A sequence or series diverges if it does not approach a finite limit. Example: The series ∑ 1/n (harmonic series) diverges.
  • Applications: Used in calculus, physics, and engineering to model phenomena like signal processing or financial calculations.

2.2 Convergence Test of Infinite Series (Statement Only)

An infinite series ∑ aₙ converges if the sequence of partial sums Sₙ = a₁ + a₂ + ... + aₙ approaches a finite limit as n → ∞. Otherwise, it diverges.

  • N-th Term Test (Divergence Test): If limₙ→∞ aₙ ≠ 0 or the limit does not exist, then ∑ aₙ diverges. Note: If limₙ→∞ aₙ = 0, the series may still diverge.
  • Example: For ∑ n/(n+1), limₙ→∞ aₙ = limₙ→∞ n/(n+1) = 1 ≠ 0, so the series diverges.

2.3 Direct Comparison Test, Limit Comparison Test (Statement Only)

  • Direct Comparison Test: Let ∑ aₙ and ∑ bₙ be series with aₙ ≥ 0 and bₙ ≥ 0. If 0 ≤ aₙ ≤ bₙ for all n sufficiently large:
    • If ∑ bₙ converges, then ∑ aₙ converges.
    • If ∑ aₙ diverges, then ∑ bₙ diverges.
    Example: Compare ∑ 1/(n² + 1) to ∑ 1/n². Since 1/(n² + 1) < 1/n² and ∑ 1/n² converges, ∑ 1/(n² + 1) converges.
  • Limit Comparison Test: Let ∑ aₙ and ∑ bₙ be series with aₙ > 0 and bₙ > 0. If limₙ→∞ (aₙ / bₙ) = L, where L is a positive finite number, then either both series converge or both diverge. If L = 0 and ∑ bₙ converges, then ∑ aₙ converges. If L = ∞ and ∑ bₙ diverges, then ∑ aₙ diverges.
  • Example: For ∑ 1/(n + n²), compare with ∑ 1/n². Compute limₙ→∞ [(1/(n + n²)) / (1/n²)] = limₙ→∞ n²/(n + n²) = 1. Since ∑ 1/n² converges, ∑ 1/(n + n²) converges.

2.4 P-series Test, D’Alembert’s Ratio Test, and Alternating Series Test (Statement Only)

  • P-series Test: The series ∑ 1/nᵖ converges if p > 1 and diverges if p ≤ 1.
    • Example: ∑ 1/n² converges (p = 2 > 1); ∑ 1/n diverges (p = 1).
  • D’Alembert’s Ratio Test: For a series ∑ aₙ with aₙ ≠ 0, compute limₙ→∞ |aₙ₊₁ / aₙ| = L:
    • If L < 1, the series converges absolutely.
    • If L > 1 or L = ∞, the series diverges.
    • If L = 1, the test is inconclusive.
    • Example: For ∑ n!/2ⁿ, |aₙ₊₁ / aₙ| = [(n+1)!/2ⁿ⁺¹] / [n!/2ⁿ] = (n+1)/2 → ∞, so the series diverges.
  • Alternating Series Test: For an alternating series ∑ (-1)ⁿ aₙ or ∑ (-1)ⁿ⁺¹ aₙ with aₙ > 0, the series converges if:
    • aₙ is decreasing (or eventually decreasing).
    • limₙ→∞ aₙ = 0.
    • Example: For ∑ (-1)ⁿ⁺¹ (1/n), aₙ = 1/n is decreasing, and limₙ→∞ 1/n = 0, so the series converges.
