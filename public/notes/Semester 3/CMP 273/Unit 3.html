Unit 3: Array and Function

3.1 Working with Array

Arrays in PHP are very flexible — they can be indexed, associative, or even multidimensional, and they can change size dynamically.

3.1.1 Indexed Array (Numeric / Sequential)

<?php
// Indexed array (keys are 0, 1, 2...)
$colors = array("Red", "Green", "Blue", "Yellow");

// Short syntax (PHP 5.4+)
$fruits = ["Apple", "Banana", "Mango", "Orange"];

echo $fruits[1];          // Banana
echo $fruits[3];          // Orange

// Add element at the end
$fruits[] = "Grapes";

// Change value
$fruits[0] = "Pineapple";

// Count elements
echo count($fruits);      // 5
?>

3.1.2 Associative Array

<?php
$student = [
    "name"    => "Ram Sharma",
    "roll"    => "075BCT001",
    "faculty" => "BCT",
    "gpa"     => 3.85,
    "active"  => true
];

echo $student["name"];     // Ram Sharma
echo $student["gpa"];      // 3.85

// Add new key-value
$student["email"] = "ram@ioe.edu.np";

// Check if key exists
if (isset($student["phone"])) {
    echo $student["phone"];
} else {
    echo "Phone not set";
}
?>

3.1.3 Array Iteration

<?php
// foreach — most common for arrays
$days = ["Sun", "Mon", "Tue", "Wed", "Thu", "Fri", "Sat"];

foreach ($days as $day) {
    echo "$day ";
}
// Output: Sun Mon Tue Wed Thu Fri Sat

// With key and value (associative)
foreach ($student as $key => $value) {
    echo "$key: $value <br>";
}

// Traditional for loop (indexed only)
for ($i = 0; $i < count($days); $i++) {
    echo $days[$i] . " ";
}
?>

3.1.4 Multi-dimensional Array

<?php
// 2D array (array of arrays)
$students = [
    ["name" => "Ram", "roll" => "001", "gpa" => 3.85],
    ["name" => "Sita", "roll" => "002", "gpa" => 3.92],
    ["name" => "Hari", "roll" => "003", "gpa" => 3.45]
];

// Access
echo $students[1]["name"];     // Sita
echo $students[0]["gpa"];      // 3.85

// Loop through 2D array
foreach ($students as $student) {
    echo $student["name"] . " - GPA: " . $student["gpa"] . "<br>";
}
?>
Best practice: Use meaningful keys in associative arrays. Avoid mixing indexed and associative in the same array level (can cause confusion).

3.3 User Defined Function

3.3.1 Passing arguments and Return

<?php
// Basic function
function greet($name) {
    echo "Hello, $name!";
}

// With return
function add($a, $b) {
    return $a + $b;
}

// Default parameter
function welcome($name = "Guest") {
    return "Welcome, $name!";
}

// Call functions
greet("Sita");             // Hello, Sita!
echo add(15, 27);          // 42
echo welcome();            // Welcome, Guest!
echo welcome("Hari");      // Welcome, Hari!
?>

3.3.2 Variable Scoping

  • Local: Defined inside function → only available inside
  • Global: Defined outside → use global keyword inside function
  • Static: Retains value between function calls
  • Parameter: Passed to function (local scope)
<?php
$globalVar = 100;           // global

function testScope() {
    $localVar = 50;         // local
    global $globalVar;      // access global
    $globalVar += 10;

    static $counter = 0;    // static - remembers value
    $counter++;
    echo "Counter: $counter <br>";
}

testScope();                // Counter: 1
testScope();                // Counter: 2
echo $globalVar;            // 110
// echo $localVar;          // Error – not accessible
?>
Modern recommendation (PHP 7.4+):
Avoid using global — pass variables as parameters or use dependency injection / classes.
Use static only when you really need to preserve state.
