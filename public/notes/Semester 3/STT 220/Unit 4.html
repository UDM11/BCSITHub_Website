Unit 4: Correlation and Regression Analysis (6 hours)

4.1 Meaning, Definition of Correlation

Correlation measures the strength and direction of the linear relationship between two quantitative variables. It tells us how changes in one variable are associated with changes in another.

Key Points

  • Correlation does NOT imply causation
  • Only measures linear relationship (not non-linear)
  • Range of correlation coefficient (r): –1 to +1
  • Interpretation of r value:
    • r = +1 → perfect positive linear correlation
    • r = 0 → no linear correlation
    • r = –1 → perfect negative linear correlation
    • |r| > 0.7 → strong correlation
    • |r| 0.4–0.7 → moderate
    • |r| < 0.4 → weak
Nepal IT context examples:
• Positive correlation: study hours vs BCA semester marks
• Negative correlation: server response time vs number of users online
• No correlation: student height vs programming skill

4.2 Karl Pearson’s Correlation Coefficient

Karl Pearson’s product-moment correlation coefficient (r) is the most widely used measure of linear correlation.

\[ r = \frac{\sum (x_i - \bar{x})(y_i - \bar{y})}{\sqrt{\sum (x_i - \bar{x})^2 \sum (y_i - \bar{y})^2}} \]

or equivalently:

\[ r = \frac{\text{Cov}(X,Y)}{s_x \cdot s_y} \]

Where:

  • Cov(X,Y) = covariance between X and Y
  • s_x, s_y = standard deviations of X and Y

Properties of Pearson’s r

  • –1 ≤ r ≤ +1
  • r = 0 → no linear association
  • Sign indicates direction (+ = direct, – = inverse)
  • Magnitude indicates strength
  • Unaffected by change of origin or scale
  • Only measures linear relationship
Small dataset example (study hours vs marks):
Hours (x): 2, 4, 5, 7, 8
Marks (y): 45, 60, 68, 82, 90 After calculation: r ≈ 0.98 (very strong positive correlation)
For grouped data: use mid-values of classes and apply same formula with frequencies.

4.3 Meaning and Definition of Simple Regression

Regression analysis is used to:

  • Understand how one variable (dependent) changes with another (independent)
  • Predict / estimate the value of dependent variable for given independent variable(s)

Simple Linear Regression (one independent variable):

\[ Y = a + bX + e \]

Where:

  • Y = dependent variable (to be predicted)
  • X = independent variable (predictor)
  • a = intercept (value of Y when X = 0)
  • b = slope / regression coefficient (change in Y per unit change in X)
  • e = error / residual term

Least Squares Method (to find best-fit line)

Minimizes sum of squared vertical distances (residuals) from points to line.

\[ b = \frac{\sum (x_i - \bar{x})(y_i - \bar{y})}{\sum (x_i - \bar{x})^2} = r \cdot \frac{s_y}{s_x} \]
\[ a = \bar{y} - b \bar{x} \]
Using previous data (hours vs marks):
b ≈ 6.5 (each extra hour → ~6.5 marks increase)
a ≈ 32.5
Regression equation: Marks = 32.5 + 6.5 × Hours

4.4 Standard Error of Estimate

Standard Error of Estimate (SEE) measures the accuracy of predictions made by the regression line — average distance of actual values from predicted values.

\[ S_e = \sqrt{\frac{\sum (y_i - \hat{y}_i)^2}{n-2}} \]

or

\[ S_e = s_y \sqrt{1 - r^2} \]
  • Smaller S_e → better fit / more accurate predictions
  • S_e has same unit as Y
  • When r = ±1 → S_e = 0 (perfect fit)
  • When r = 0 → S_e = s_y (no linear relationship)
In IT: Used to evaluate regression models (e.g., predicting server load, response time, sales forecast accuracy).

4.5 Coefficient of Determination

Coefficient of Determination (r²) tells us what proportion of the variation in the dependent variable (Y) is explained by the independent variable (X) in the regression model.

\[ r^2 = 1 - \frac{\sum (y_i - \hat{y}_i)^2}{\sum (y_i - \bar{y})^2} = \left( \frac{\text{Cov}(X,Y)}{s_x s_y} \right)^2 \]
  • 0 ≤ r² ≤ 1
  • r² = 0 → X explains 0% of variation in Y
  • r² = 1 → X explains 100% of variation in Y
  • r² × 100% = percentage of variance in Y explained by X
If r = 0.98 (from earlier example):
r² = (0.98)² = 0.9604 → 96.04% of variation in marks is explained by study hours
Remaining 3.96% due to other factors (sleep, IQ, teaching quality…)
Important interpretation tip (common exam question):
“r² = 0.64” means → 64% of the variation in the dependent variable is explained by the independent variable. The remaining 36% is due to other variables or random error.
In IT context: r² is widely used to evaluate predictive models (e.g., how well server CPU usage predicts response time, how well code complexity predicts number of bugs).
