2.1 Construction of GFD, Relative frequency, Cumulative frequency distribution

Raw data (ungrouped observations) is difficult to understand. Frequency distributions summarize and organize data for easier analysis.

Types of Frequency Distributions

Type Meaning How to Construct Use
Grouped Frequency Distribution (GFD) Data grouped into classes/intervals 1. Find range = max – min
2. Decide number of classes (Sturges’ rule: k ≈ 1 + 3.322 log₁₀ n)
3. Class width = range / k (usually round up)
4. Make class intervals (exclusive or inclusive)
5. Tally frequencies 		Large data sets, histograms, analysis
Relative Frequency Distribution Proportion or percentage of total observations in each class Relative freq = (class frequency) / total observations Compare distributions of different sizes
Cumulative Frequency Distribution Running total of frequencies (less than or more than type) Cumulative freq = sum of frequencies up to that class Ogive graphs, percentiles, median from graph

Step 2: No. of classes ≈ 1 + 3.322 log₁₀(40) ≈ 6–7 (take 6)

2.1 Measure of Central Tendency

Measures of central tendency give a single value that represents the “center” or typical value of the data set.

Measure Definition Formula / Method Best Used When Sensitive to Extreme Values?
Arithmetic Mean (Average) Sum of values divided by number of values μ = Σx / n (ungrouped)
x̄ = Σ(f×x) / Σf (grouped)		 Symmetrical data, interval/ratio scale Yes (very sensitive)
Median Middle value when data ordered n odd → (n+1)/2 th item
n even → average of n/2 and (n/2)+1 th items		 Skewed data, ordinal scale, outliers present No
Mode Most frequent value Highest frequency class (grouped: modal class) Nominal data, categorical data, multimodal No
Geometric Mean nth root of product of n values GM = (x₁ × x₂ × … × xₙ)^(1/n) Growth rates, ratios, percentages Yes
Harmonic Mean n divided by sum of reciprocals HM = n / Σ(1/xᵢ) Rates, speeds, time per unit distance Yes
Data: 12, 15, 18, 20, 22, 25, 30, 100
Mean = 30.25 (pulled up by 100)
Median = (20 + 22)/2 = 21 (not affected by 100)
Mode = no mode (all unique)

2.2 Measure of Location

Measures of location (also called quantiles or positional averages) divide the data into equal parts.

Main Measures

Measure Divides data into Calculation (ungrouped) Use
Median (Q₂) 2 equal parts (50%) (n+1)/2 th value Middle value
Quartiles (Q₁, Q₃) 4 equal parts Q₁ = (n+1)/4 th value
Q₃ = 3(n+1)/4 th value		 Spread in lower & upper half
Deciles (D₁ to D₉) 10 equal parts Dₖ = k(n+1)/10 th value Income distribution, test scores
Percentiles (P₁ to P₉₉) 100 equal parts Pₖ = k(n+1)/100 th value Standardized tests (e.g., percentile rank)
Interquartile Range (IQR) = Q₃ – Q₁
Used to detect outliers: any value < Q₁ – 1.5×IQR or > Q₃ + 1.5×IQR is outlier.

2.2 Measure of Dispersion

Measures of dispersion show how spread out or scattered the data values are around the central value.

Measure Definition / Formula Advantages Disadvantages Unit
Range Maximum – Minimum Very easy to calculate Only uses two values, affected by outliers Same as data
Quartile Deviation (QD) (Q₃ – Q₁)/2 Less affected by extremes Ignores 50% of data Same as data
Mean Deviation (MD) Σ|x – mean| / n Uses all values Absolute value – not algebraically convenient Same as data
Variance (σ²) Σ(x – mean)² / n (population)
Σ(x – x̄)² / (n–1) (sample)		 Uses all values, algebraic properties Square unit (not intuitive) Square of data unit
Standard Deviation (σ) √Variance Same unit as data, most widely used Sensitive to outliers Same as data
Coefficient of Variation (CV) (σ / mean) × 100% Relative measure – compares dispersion across different units Meaningless if mean ≈ 0 Percentage
Data: 10, 12, 15, 18, 20
Mean = 15
Range = 20 – 10 = 10
Variance = 14 (sample)
Std. Dev. ≈ 3.74
CV ≈ 24.93%
Rule of thumb:
• For normal distribution → ~68% data within ±1 SD
• ~95% within ±2 SD
• ~99.7% within ±3 SD
