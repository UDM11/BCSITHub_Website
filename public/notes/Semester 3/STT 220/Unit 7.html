Unit 7: Hypothesis Testing (12 hours)

7.1 Meaning and Definition

Hypothesis testing is a formal procedure in inferential statistics used to decide whether sample data supports or contradicts a claim (hypothesis) about a population parameter. We start with two competing statements and use sample evidence to choose between them.

Key Terms

  • Null hypothesis (H0): The statement we assume to be true initially; usually represents “no effect,” “no difference,” or “status quo.”
  • Alternative hypothesis (H1 or Ha): The statement we want to support; represents “there is an effect,” “there is a difference,” or a specific direction (greater than / less than).
  • Test statistic: A number computed from the sample that measures how far the sample result is from what we would expect if H0 were true (e.g., z, t, F).
  • p-value: The probability of observing a test statistic as extreme as (or more extreme than) the one we got, if H0 were true. Small p-value → evidence against H0.
  • Conclusion: If p-value ≤ significance level (α), we reject H0; otherwise we fail to reject H0.
We never “accept” H0; we either reject it or fail to reject it. Failing to reject means we do not have enough evidence to support H1.

7.2 Types of Errors, Significance Level, Degree of Freedom

Types of Errors

Decision H0 is true H0 is false
Reject H0 Type I Error (α) — False positive; reject a true null Correct — True positive
Fail to reject H0 Correct Type II Error (β) — False negative; fail to reject a false null
  • Type I error (α): Rejecting H0 when it is actually true. We control this with the significance level.
  • Type II error (β): Failing to reject H0 when it is actually false. Power = 1 − β (probability of correctly rejecting a false H0).

Significance Level (α)

The significance level (α) is the maximum probability of committing a Type I error we are willing to allow. Common choices: α = 0.05 (5%), 0.01 (1%), 0.10 (10%). We reject H0 if p-value ≤ α.

Degree of Freedom (df)

Degrees of freedom (df) is the number of independent pieces of information in the sample used to compute a statistic. It affects the shape of the t and chi-square (and F) distributions.

  • One-sample t-test (mean): df = n − 1
  • Two-sample t-test (independent): df = n1 + n2 − 2 (or use Welch’s formula if variances unequal)
  • Paired t-test: df = n − 1 (n = number of pairs)
  • One-way ANOVA (k groups, total n): dfbetween = k − 1, dfwithin = n − k

7.3 Steps in Hypothesis Testing

  1. State H0 and H1 in terms of the population parameter (μ, p, etc.).
  2. Choose the significance level α (e.g., 0.05).
  3. Select the appropriate test and test statistic (z, t, F, etc.) based on parameter, sample size, and whether σ is known or unknown.
  4. Compute the test statistic from the sample data.
  5. Find the p-value (or compare test statistic to critical value) using the sampling distribution.
  6. Decide: If p-value ≤ α, reject H0; otherwise fail to reject H0.
  7. Conclusion: State in context (e.g., “There is sufficient evidence at the 5% level to support that the mean is greater than 50”).
One-tailed vs two-tailed: Two-tailed H1: μ ≠ μ0 (use both tails for p-value). One-tailed: μ > μ0 or μ < μ0 (use one tail only).

7.4 Single and Double Mean Test (Population Standard Deviation Known)

When the population standard deviation σ is known, we use the z-test for means (based on the normal distribution).

Single mean (one-sample z-test)

Test H0: μ = μ0 vs H1: μ ≠ μ0 (or > or <).

z = (x̄ − μ0) / (σ / √n)

Compare z to critical values from the standard normal table, or find the p-value.

Two means (two-sample z-test, independent samples)

Test H0: μ1 = μ2 vs H1: μ1 ≠ μ2 (or > or <). Assume populations are normal with known σ1, σ2.

z = (x̄1 − x̄2) / √(σ1²/n1 + σ2²/n2)
Single mean: Claim μ = 100, σ = 15, n = 36, x̄ = 105. z = (105 − 100) / (15/√36) = 5/2.5 = 2. For two-tailed α = 0.05, critical value ≈ ±1.96. Since 2 > 1.96, reject H0.

7.5 Single and Double Proportion Test

For proportions, we use the z-test (large samples: np ≥ 5 and n(1−p) ≥ 5).

Single proportion

Test H0: p = p0 vs H1: p ≠ p0 (or > or <).

z = (p̂ − p0) / √[p0(1−p0) / n]

Two proportions (independent samples)

Test H0: p1 = p2 vs H1: p1 ≠ p2 (or > or <). Use pooled estimate p̄ = (x1 + x2) / (n1 + n2) under H0.

z = (p̂1 − p̂2) / √[p̄(1−p̄)(1/n1 + 1/n2)]
p̂1 = x1/n1, p̂2 = x2/n2; x1, x2 are numbers of successes in each sample.

7.6 Single and Double Mean Test (Population Standard Deviation Unknown)

When σ is unknown, we use the t-test (sample standard deviation s and t-distribution).

Single mean (one-sample t-test)

Test H0: μ = μ0 vs H1: μ ≠ μ0 (or > or <). df = n − 1.

t = (x̄ − μ0) / (s / √n)

Two means (two-sample t-test, independent samples)

Test H0: μ1 = μ2 vs H1: μ1 ≠ μ2 (or > or <). Assume roughly equal population variances (pooled) or use Welch’s t (unequal variances).

Pooled variance: sp² = [(n1−1)s1² + (n2−1)s2²] / (n1 + n2 − 2)

t = (x̄1 − x̄2) / (sp √(1/n1 + 1/n2)) , df = n1 + n2 − 2

7.7 Paired t-Test

The paired t-test is used when we have two related measurements on the same subjects (e.g., before/after, matched pairs). We work with the differences d = (after − before) and test whether the mean difference μd is zero.

Hypotheses

H0: μd = 0 vs H1: μd ≠ 0 (or > 0 or < 0)

Test statistic

d̄ = sample mean of differences, sd = sample standard deviation of differences, n = number of pairs.

t = d̄ / (sd / √n) , df = n − 1
Example: Reaction time before and after training for 10 users. Compute d = after − before for each; then d̄ and sd. Test H0: μd = 0. If t exceeds the critical value (or p-value < α), we conclude there is a significant change.
Use paired t when observations are naturally paired (same subject, matched units). Use two-sample t when the two groups are independent.

7.8 One-Way ANOVA

One-way ANOVA (Analysis of Variance) tests whether the means of three or more independent groups are equal. It extends the two-sample t-test to k groups.

Hypotheses

H0: μ1 = μ2 = … = μk (all group means are equal)
H1: At least one mean is different

Idea

Partition total variation into between-group variation (due to group differences) and within-group variation (random error). If between-group variation is large relative to within-group, we reject H0.

Test statistic: F-ratio

F = MSbetween / MSwithin

Where MS = Mean Square = SS/df.

  • SSbetween (or SSB): Sum of squares between groups
  • SSwithin (or SSW): Sum of squares within groups (error)
  • dfbetween = k − 1, dfwithin = n − k (n = total sample size, k = number of groups)

Decision

Compare F to the critical value from the F-distribution with (k−1, n−k) degrees of freedom at level α. If F > Fcritical (or p-value ≤ α), reject H0 — at least one group mean differs.

ANOVA only tells us that not all means are equal; it does not say which groups differ. For that we use post-hoc tests (e.g., Tukey’s HSD) after a significant F.
